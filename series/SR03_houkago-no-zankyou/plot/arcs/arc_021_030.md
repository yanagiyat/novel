# アーク概要

東京との導線が開通し、湊は受動的な調査者から能動的な継承者に変わる。関係者一人ひとりに向き合い、学校最後の公開放送で全員が自分の言葉を言い直す。声の地図が完成し、湊は記録を運ぶ者として東京へ出発する。

**リズム**: 終盤は謎解きだけでなく、言い直しのドラマに重心を移す

# キャラクターの変化

- **湊**: 受動→能動。調査者→継承者→出発する者
- **澪**: 他人の夢を漏らす受動的な存在→自分の言葉を自分で選んで話す人物
- **奈央**: 音を消した隠蔽者→曖昧さを含んだまま音を残す責任を引き受ける
- **颯真**: 取り繕う人気者→自分の卑小さを認めたうえで前に出る
- **千紘**: 善意ゆえに自分を罰する→限界と罪を引き受けてなお手を伸ばせる
- **琴葉**: 守るための沈黙→残された者たちに語らせる方向へ
- **桐野**: 逃げていた大人→責任を引き受け若者の出発を支える

# メモ

- EP029が最大のカタルシス。公開放送で全員が自分の言葉を言い直す
- EP030は寂しさと前進が同居する締め。湊の出発は逃避ではなく継承
- 超常現象の完全な種明かしはしない。余韻を残す
- 第一部としての事件は閉じるが、声の旅は続く余白を残す
