# アーク概要

放課後の校内放送に死んだはずの美空の声が混じるところから物語が始まる。湊は一年前の事故を追い始め、能力者たちの異変に触れ、『声の地図』の存在を知り、事故が「ただの事故ではない」という仮説に到達する。

**リズム**: 毎話に小さな発見を置く。日常の空気にじわじわ異常が混ざる

# キャラクターの変化

- **湊**: 偶然の気づき→意図的な調査者へ。まだ受動的
- **澪**: 異変を隠す日常を見せる
- **奈央**: 防御的な姿勢。協力を拒む
- **颯真**: 明るい仮面の下の影がちらつく
- **千紘**: 善意と恐怖の板挟みを示す
- **琴葉**: 統制者として情報を管理する側で登場

# メモ

- EP001は静かに始める。放送に混じった声は数秒だけ。大半の生徒は気づかない
- 能力描写は最初から名称化しない。異変として描く
- EP010で第一段階の仮説が立つが、まだ全貌は見えない
