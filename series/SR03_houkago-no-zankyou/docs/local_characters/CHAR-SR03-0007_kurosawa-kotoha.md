# 概要

高校3年、女子。生徒会副会長。美空の親友であり、『声の地図』の企画意図を最も深く理解している人物。有能で責任感が強く、感情を整理してから話す。死後の美空を守るつもりで多くを隠しているが、その「守り方」は他人の選択権を奪う形になっている。

# 外見

- しっかりして見える安心感
- 姿勢が良い
- 人前で崩れない
- 笑うときも気を配っている

# 能力

基本は無能力者。超常的な異変は持たない。

# 人格

- 有能
- 責任感が強い
- 感情を整理してから話す
- 周囲から信頼される
- 自分が抱えれば回ると思ってしまう
- 他人の選択権を奪う形で守ろうとする

# 秘密

- 美空のノートや企画書の一部を保管している
- 美空の企画を止めるべき瞬間に止められなかった
- 誰にも見せていない封筒を持っている

# 対人関係

- **湊（CHAR-SR03-0001）**: 協力者のように見えて情報を選んで出す。湊は彼女を頼りながらも最も危うい語り手だと気づく
- **美空（CHAR-SR03-0002）**: 親友。理解者であり制止役だったが、止めきれなかった
- **奈央（CHAR-SR03-0004）**: 美空の死後処理をめぐる結託とひび

# キャラクターアーク

守るための沈黙を続けていた人物が、本人の代わりに語るのではなく、残された者たちに語らせる方向へ変わる。
