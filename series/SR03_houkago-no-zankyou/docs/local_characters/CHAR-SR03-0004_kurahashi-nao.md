# 概要

高校2年、女子。放送部所属。事故の夜以降、自分の近くで録音された音声に、その場で誰かが飲み込んだ未発言の一言がノイズのように混じることがある。理詰めで几帳面、感情をあまり見せないが内側は繊細。一度誤ると自分を許せない性格。

# 外見

- 冷たく見える整然さ
- 髪や制服に無駄がない
- 機材を持つ手つきは慣れている
- 目の下に寝不足の気配が出やすい

# 異変：「未発言ノイズ」

自分の近くで録音された音声に、その場で誰かが飲み込んだ未発言の一言がノイズのように混じることがある。

**制限**:
- 誰の言葉かわからない
- 本心か衝動かわからない
- 常に発生するわけではない

**代償**:
- 録音の信頼性への崩壊
- 放送部員としての自己否定
- 大事な音源を消した過去への罪悪感

# 人格

- 理詰め
- 几帳面
- 感情をあまり見せない
- 内側は繊細
- 信頼するまで時間がかかる
- 一度誤ると自分を許せない
- 冷たさで先回りして自衛する

# 家庭背景

期待に応えることを当然とする家庭。失敗を隠したくなる背景を持つ。

# 秘密

- 一年前の重要な音声を自分の判断で消した
- その音声が誰かを不当に追い詰めると思った、あるいは美空の意図を誤解させる危険があると思った
- **EP004で湊に告白した内容**:
  - 事故後、美空のPC内の音声ファイルとSDカードのバックアップを全て消去した
  - 消去の理由: 「守りたかったから」（誰を守ったかは未回答）
  - MDの素材テープは古すぎてPCに取り込んでいなかったため消去を免れた
  - 素材⑧のMDは消し忘れ（存在を忘れていた）
  - 美空の善意が人の境界線を越えることへの複雑な感情:「いい人は、ときどき怖い。善意で人の境界線を越える」

# 対人関係

- **湊（CHAR-SR03-0001）**: 湊の聞く姿勢に少しずつ信頼を置く
- **美空（CHAR-SR03-0002）**: 才能と勢いに圧倒されつつ技術面で支えた。尊敬と反発が混在
- **琴葉（CHAR-SR03-0007）**: 美空の死後処理をめぐる結託とひび。守るために隠すか、残すために曖昧さを引き受けるか

# キャラクターアーク

何かを守るために音を消した人物から、あいまいさを含んだまま音を残す責任を引き受ける人物へ変わる。
