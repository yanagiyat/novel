# 概要

高校2年、女子。陸上部兼保健委員。面倒見がよく責任感が強い。事故の夜以降、人に触れるとその人が隠している痛みや苦しさが自分の身体に数秒だけ流れ込むことがある。助けたいのに触れられない矛盾を抱えている。

# 外見

- 健康的でしっかりして見える
- 運動で鍛えた身体
- 普段は明るめだが、痛みの発作時に顔色が変わる

# 異変：「痛覚流入」

人に触れると、その人が隠している痛みや苦しさが自分の身体に数秒だけ流れ込むことがある。外傷だけでなく、息苦しさ、吐き気、古傷、殴られた痛みなども含む。

**制限**:
- 痛みの意味や原因は断定できない
- 本人の記憶ではなく身体感覚だけ
- 避けようとしても偶発的に起こる

**代償**:
- 他人に触れること自体が怖くなる
- 助ける行為が自傷に近づく
- 善意が続かなくなる恐怖

# 人格

- 面倒見がよい
- 責任感が強い
- 誰かを見捨てるのが苦手
- 頑張りすぎる
- 助けられなかった自分を罰する
- 弱音が遅い

# 家庭背景

家庭内で「しっかり者」を担っている。多少無理をしても平気なふりをする癖がある。

# 秘密

- 美空の状態にいったん近づいたのに、最適な行動を即座に取れなかった
- 優しい人でいたいのに、それが壊れたと感じている

# 対人関係

- **湊（CHAR-SR03-0001）**: 比較的正直だが、自分の罪に関わる部分では黙る
- **美空（CHAR-SR03-0002）**: 善意の共鳴。助ける側の人間として惹かれ合うが、実際には互いに無理をしていた
- **颯真（CHAR-SR03-0005）**: 事故の夜の共有記憶があり、互いに見たくないものを知っている

# キャラクターアーク

善意ゆえに自分を罰していた人物が、自分の限界と罪を引き受けたうえでなお他者に手を伸ばせるようになる。
