# 概要

高校2年、男子。新聞部所属。超常的な能力は一切持たない。静かで観察力が高く、人の話を遮らない。引っかかったことを放置できない性格。以前、有馬美空のインタビュー音源の文字起こしを少し手伝ったことがあり、完全な部外者ではない。美空の死後、放課後の校内放送に彼女の声が混じったことをきっかけに、一年前の事故を追い始める。

# 外見

- 華奢ではないが目立たない体格
- 整いすぎず、印象に残りにくい顔
- 黒髪、やや伸び気味、無造作
- 視線が強すぎず、相手に話させるタイプ
- 制服を崩しすぎない。実用本位
- クラスでは背景に溶けるが、会話のあとで記憶に残る

# 口調

- 一人称：「俺」（独白）/「僕」（対外的に使い分ける場面も可）
- 質問が多いが、詰問にはならない
- 沈黙を挟むことが多い
- 感情を言語化するのが遅い

# 能力・技能

- **無能力**。超常的な力は一切ない
- **聞く力**: 人の話を遮らず、細部を覚える。相手に話させる空気を作れる
- **記録者としての資質**: 新聞部で鍛えた取材メモの正確さ。事実と印象を分けて記録する
- **欠点**: 自分の感情を言語化するのが遅い。他人の事情を背負い込みやすい。優柔不断に見える

# 内面の葛藤

暴くことと聞くことの違いをまだ自分の中で整理できていない。美空の残したものを守りたい気持ちと、自分がそれを受け継ぐ資格があるのかという迷いのあいだで揺れる。

# 秘密

- 美空の死後、彼女の声が自分の記憶の中で少し美化されている自覚がある
- 事件を追う理由に、真実だけでなく、美空の企画を終わらせたくない私情がある

# 家庭背景

母は現実的で、地元に残る進路を否定しない。家庭は壊れてはいないが、将来について熱く話す文化ではない。自分から大きな夢を口にする習慣が薄い。

# 対人関係

- **有馬美空（CHAR-SR03-0002）**: 不在の先輩。文字起こしの記憶や音源を通じて対話を続けるような関係。美化と現実認識の差が湊の成長に関わる
- **早瀬澪（CHAR-SR03-0003）**: 幼なじみ。安心と緊張が同居する。近すぎて本音を言いにくい
- **倉橋奈央（CHAR-SR03-0004）**: 協力と不信。奈央の消したものを疑うが、守ろうとしたものにも気づく
- **深澤颯真（CHAR-SR03-0005）**: 距離のある同学年。調査の中で颯真の内側の脆さに触れる
- **白崎千紘（CHAR-SR03-0006）**: 信頼の再構築。千紘の善意の重さを理解していく
- **黒沢琴葉（CHAR-SR03-0007）**: 情報の供給者と統制者。頼りながらも最も危うい語り手だと気づく
- **桐野直人（CHAR-SR03-0008）**: 教師の弱さを見抜くが単純な敵にはできない
- **真鍋新（CHAR-SR03-0009）**: 終盤から関わる。東京への現実的な橋

# キャラクターアーク

1. **初期（EP1-10）**: 美空の声をきっかけに一年前の事故を追い始める受動的な調査者
2. **中期（EP11-20）**: 全員が少しずつ加害者でもある構図に到達。自分の調査の危うさにも気づく
3. **転換（EP21-22）**: 真相を掘るだけでなく、声をどう残すかを考え始める。受動から能動へ
4. **最終（EP23-30）**: 『声の地図』を再構成し、関係者に声を言い直させる場を作る。東京へ出発
