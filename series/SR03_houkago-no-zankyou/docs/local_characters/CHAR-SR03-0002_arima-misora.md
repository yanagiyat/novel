# 概要

当時高校2年、女子。放送部所属。一年前の旧特別棟停電事故で転落死。明るく人懐っこく、他人の声に価値を見出す人物。匿名音声ドキュメンタリー『声の地図』の企画者。善意がある一方で踏み込みすぎる面があり、自分の行為が相手にどう見えるかに鈍いことがあった。

# 外見

- 明るく目を引く。笑うと場の温度が少し上がる
- 軽く結ぶことが多い髪
- よく通る声。威圧感ではなく親密さがある
- 録音機材を抱えて歩く姿が似合う

# 人格

- 人懐っこい
- 好奇心が強い
- 他人の声に価値を見出す
- 善意がある
- 踏み込みすぎる
- 自分の行為が相手にどう見えるかに鈍いことがある

## 矛盾

人の痛みに寄り添いたいと思っているが、そのために相手の境界を越えてしまう。救いたい気持ちと聞き出したい欲求が混ざっている。

# 進路志望

音声ドキュメンタリーを作る仕事や音声アーカイブに関わる方向。真鍋新と連絡を取り、作品を東京のユース企画へ出す可能性を探っていた。

# 事故における役割

『声の地図』最終素材の整理と、関係者との対話の最中に停電事故へ巻き込まれる。機材バックアップまたは確認のため上階へ向かい、転落。

# 他者からの誤読

- 秘密を暴こうとしている
- 人の弱みを握っている
- 善意のふりをした支配的な人間かもしれない

# 実際

暴露自体を目的にはしていないが、結果的にそう誤解されても仕方がない振る舞いをしていた。

# 禁止事項

- 完全な聖人として描かないこと
- 善意と危うさの両面を常に持たせること

# 登場履歴

## EP003

- 素材⑧（10月21日録音、北口商店街取材）の末尾に、美空が誰かに対して「ごめんなさい。勝手に聞いてしまって」と謝り、「これは残さなきゃいけないと思うの。あなたの声も——」と語りかけている断片が発見される
- 通常の商店街取材パートとは明らかに異なる場所（狭い室内、コンクリートの壁）で録音されている
- 相手は無言のまま録音が途切れる
- 美空が事故前に、取材の範囲を超えて誰かの秘密や痛みに踏み込んでいたことを示す音声的証拠
