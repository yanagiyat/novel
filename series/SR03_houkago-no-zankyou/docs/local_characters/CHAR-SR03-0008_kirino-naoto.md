# 概要

20代後半、男性。県立星原高校教諭で、新聞部と放送部の顧問を兼務。生徒思いだが保身もある。旧特別棟の非常灯や設備不良を以前から把握していたが、強く対応を進めなかった。事故当夜も報告や判断が遅れた。

# 外見

- 若いが少し疲れて見える
- ネクタイを緩めがち
- ちゃんとしようとして中途半端になる

# 人格

- 面倒見は悪くない
- 真面目
- 責任の重さに弱い
- 現場では優しいが制度の前で縮む
- 先送り
- 責任回避
- 言い訳の整合性を先に考えてしまう

# 秘密

- 設備不良に関する記録や申請を曖昧に処理していた
- 生徒に慕われたい気持ちと、責任を取りたくない気持ちが同居している

# 対人関係

- **湊（CHAR-SR03-0001）**: 後ろめたさと期待を抱く。湊は桐野の弱さを見抜くが単純な敵にはできない

# キャラクターアーク

生徒のためと言いながら逃げていた大人が、最後には自分の責任を引き受け、若者の出発を支える側に回る。
