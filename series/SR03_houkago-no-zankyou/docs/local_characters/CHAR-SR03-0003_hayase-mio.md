# 概要

高校2年、女子。美術部所属。湊の幼なじみ。事故の夜以降、眠りに落ちる直前に近くにいる人間の夢の断片を寝言として口にしてしまう異変が生じている。本人は制御できない。感受性が強く言葉少なめ。本当は依存心が強いが、傷つく前に距離を取る傾向がある。

# 外見

- 静かで、少し距離のある美しさ
- 手に絵の具や鉛筆の跡が残りやすい
- 制服はきちんとしているがどこか気が抜けている
- まぶしい場所より曇り空が似合う

# 異変：「夢漏れ」

眠りに落ちる直前だけ、近くにいる人間の夢の断片を寝言として口にしてしまうことがある。

**制限**:
- 意識的に使えない
- 夢は事実そのものではない
- 聞いた側が誤解しやすい

**代償**:
- 人を傷つけることへの恐怖
- 人前で眠れない
- 親密さを避ける

# 人格

- 感受性が強い
- 言葉少なめ
- 自分の内側を見せるのが苦手
- 人を観察する
- 本当は依存心が強い
- 傷つく前に距離を取る
- 曖昧な優しさで誤魔化す

# 家庭背景

家庭は表面上穏やかだが、会話が少なく、それぞれが自分の場所に閉じている。家庭内の沈黙への慣れが、彼女の性格に影響している。

# 秘密

- 事故の夜、眠る直前に誰かの決定的な夢の断片を口にしてしまった可能性がある
- 美空を嫌ってはいなかったが、少し羨ましく、少し怖かった

# 対人関係

- **湊（CHAR-SR03-0001）**: 幼なじみ。近すぎて本音を言えない時期が続いている。安心でもあり試される相手でもある
- **美空（CHAR-SR03-0002）**: 引力と警戒。美空に惹かれつつ、自分の境界に踏み込まれる怖さも感じていた

# 恋愛軸

湊との関係は友情と恋情のあいだで揺れるが、主題にはしない。

# キャラクターアーク

他人の夢を漏らしてしまう受動的な存在から、自分の言葉を自分で選んで話す人物へ変化する。
