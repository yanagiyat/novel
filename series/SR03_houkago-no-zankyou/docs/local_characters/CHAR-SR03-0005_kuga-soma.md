# 概要

高校2年、男子。サッカー部所属。社交的で空気を読み、場を回すのがうまい人気者。しかし事故の夜以降、誰かが迷った瞬間にその人が選ばなかったもう一つの動きの残像が見えるようになった。人の後悔ばかりが目につき、自分自身のあの数分を繰り返し見ている。明るさは自己防衛になっている。

# 外見

- 陽に当たる場所が似合う
- 運動部らしい体格
- 笑顔の圧がある
- グラウンドでは絵になるが、一人のときだけ影が濃い

# 異変：「選択残像」

誰かが迷った瞬間、その人が選ばなかったもう一つの動きの残像が見えることがある。助けようとしてやめた、言おうとしてやめた——そのような捨てられた選択だけが見える。

**制限**:
- 見えるのは現実に起きなかった行動
- それが最善だったかはわからない
- いつ見えるか制御できない

**代償**:
- 人の後悔ばかりが目につく
- 自分自身のあの数分を繰り返し見る
- 明るさが自己防衛になる

# 人格

- 社交的
- 空気を読む
- 場を回すのがうまい
- 弱みを見せない
- 本当は自己嫌悪が強い
- 体面を優先してしまう
- 人前の自分から降りられない

# 家庭背景

地元で期待される「ちゃんとした子」の役割を背負っている。部活や進路でも周囲の期待が重い。

# 秘密

- 救助よりも先に、自分の立場や見え方を守る行動を取った数分がある
- 自分は善人だと思っていた自己像が壊れている

# 対人関係

- **湊（CHAR-SR03-0001）**: 接点は薄かったが、調査の中で湊を鏡のように感じる
- **美空（CHAR-SR03-0002）**: 表面的にはそこまで親しくないが、事故の夜に短い接点があった
- **千紘（CHAR-SR03-0006）**: 表向きは普通の同級生だが、事故の夜の共有記憶があり、互いに見たくないものを知っている

# キャラクターアーク

取り繕う人気者から、自分の卑小さを認めたうえで前に出る人物へ変わる。
