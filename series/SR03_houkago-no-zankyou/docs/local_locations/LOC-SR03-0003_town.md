# 概要

若者は外に出ていくのが前提で、大人はそれを止められない。商店街は細く続き、駅前は整備されたが空白も多い。大きな事件は少ないが、小さな諦めが町じゅうに積もっている。

# 雰囲気

- 静かな閉塞
- 生活感
- 見慣れた風景の中の違和感

# 地域方針

地域名・都道府県名は明示しない。どこにでもありそうな地方都市として設計。町にはそれを受け止め保存する制度や余力がない。
