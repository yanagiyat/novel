# 概要

取り壊し予定の旧校舎。かつて放送部や地域コミュニティFMで使われていた。旧式の録音機材、カセット、MD、ICレコーダー、卓上ミキサー、放送原稿束が残る。過去の声が物理的に蓄積された場所であり、停電事故の現場。

# 象徴的役割

消えかけた記録の倉庫であり、物語の記憶装置。学校統廃合と撤去により、残る音源やテープや原稿は廃棄の危険にある。

# 重要な場所

| 場所 | 機能 |
|------|------|
| 旧特別棟機材室 | 一年前の音源・原稿・物証が眠る場所 |
| 旧特別棟上階と階段 | 事故の移動経路と転落の背景 |

# 事故との関連

- 設備不良（非常灯・連絡手段が不十分）
- 落雷による停電で真っ暗になった
- 雨音と混乱で状況把握が遅れた
- 美空は機材バックアップまたは確認のため上階へ向かい転落
