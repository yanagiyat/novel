# 概要

統廃合が決まった古い県立高校。一年後には消える学校であり、最後の一年だけ妙に濃い。進路、部活、恋愛、教師の異動、町の衰退が重なっている。

# 雰囲気

- 一年後には消える学校
- 最後の一年だけ妙に濃い
- 夕方のチャイムと窓の白い光が印象的

# 校内の重要な場所

| 場所 | 機能 |
|------|------|
| 放送室 | 現在進行の校内放送、機材トラブル、死者の声の発端 |
| 新聞部部室 | 主人公の思考と記録の拠点 |
| 美術部室 | 早瀬澪との静かな対話の場所 |
| 保健室 | 白崎千紘の能力が身体症状として表れやすい場所 |
| 体育館裏 | 秘密の対話、逃げ場のない会話 |
| 誰もいないグラウンド | 深澤颯真の孤独と残像のイメージ |
| 雨の廊下 | 停電事故の記憶と重なる導線 |
