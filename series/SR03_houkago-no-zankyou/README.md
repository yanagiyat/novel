# SR03: 放課後の残響

## ジャンル
青春群像劇 / 学園ミステリ / 微量の超常

## コンセプト
統廃合が決まった地方の県立高校で、一年前に起きた放送部員の死亡事故を、無能力の男子高校生が、壊れかけた能力者たちの証言と沈黙をつなぎ合わせて追っていく。超能力は真実を暴くための便利な力ではなく、人間関係を壊し、後悔を増幅させる不具合のような現象。最後には、失われるはずだった声と記録を主人公が受け継ぎ、東京へ運ぶ決断をする。

## 文体
パターン4（抒情・エモーショナル文体）ベース。三人称限定視点を話ごとに切り替える群像スタイル。
説明しすぎない。会話、沈黙、仕草、場所の手触りで見せる。超常は小さな不具合として混ぜる。
恋愛は補助線にとどめ、各人物に責められる理由と責めきれない理由を持たせる。

## 主要キャラクター

- **柊 湊**（高2・男）: 新聞部。無能力。観測者・聞き手・記録者。事件を追い、声を運ぶ者になる
- **有馬 美空**（死亡）: 元放送部。『声の地図』の創始者。不在の中心人物
- **早瀬 澪**（高2・女）: 美術部。湊の幼なじみ。「夢漏れ」——眠り際に他人の夢の断片を口にする
- **倉橋 奈央**（高2・女）: 放送部。「未発言ノイズ」——録音に飲み込まれた言葉が混じる
- **深澤 颯真**（高2・男）: サッカー部。「選択残像」——選ばれなかった行動の残像が見える
- **白崎 千紘**（高2・女）: 陸上部兼保健委員。「痛覚流入」——触れると隠された痛みが流れ込む
- **黒沢 琴葉**（高3・女）: 生徒会副会長。美空の親友。無能力。最も都合の悪い事実を握る
- **桐野 直人**（20代後半・男）: 新聞部・放送部顧問。生徒思いだが保身もある大人の代表
- **真鍋 新**（30代・男）: 東京の民間音声アーカイブ施設勤務。東京への導線

## 中心事件

**旧特別棟停電事故**（物語開始の一年前）
- 落雷を伴う悪天候による停電の夜、有馬美空が旧特別棟で転落死
- 完全な事故ではあるが、単純な偶然でもない
- 救助があと10分早ければ助かった公算が高い
- その10分の遅れは、一人の殺意ではなく、複数の人物の迷い・保身・誤解・恐怖・沈黙が重なって生まれた

## P25ML共有世界との接点

- world_exposure: none — 裏設定はほぼ出さない。単独の青春群像劇として成立
- 超常現象はP25MLの「超能力」系統に分類可能だが、作中では体系化しない
- 舞台は意図的に地名を明示しない地方都市。既存舞台とは別
- 共有インフラ企業への言及は不要

## 執筆時の注意

- 犯人探しを目的化しない。各人物がなぜ黙ったのか、なぜ遅れたのかを再構成する物語
- 美空を完全な聖人として描かない。善意と踏み込みすぎる危うさの両面を持たせる
- 主人公を万能な探偵にしない。暴くことと聞くことの違いを彼自身も模索している
- 超常は最後まで確定させない。超常か錯覚かを読者が揺れ続ける設計
- 地方をただの閉塞の象徴にしない。生活感と寂しさと前進を同居させる
- ラストの上京は夢追いではなく記録の継承。逃避ではなく引き受けること
