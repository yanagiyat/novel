# 概要

34歳、女性。元警視庁捜査一課刑事。担当事件の証拠に不審を抱き組織内で孤立した末に退職。2年前、半ば逃げ込む形で槻ノ森市に移住し、私立探偵事務所を開業した。論理と物証だけを信じる。「説明できないことは存在しない」が信条。

（執筆時に確定。鋭さはあるが華美ではない。元刑事の実務的な佇まい）

# 口調

- 一人称：「私」
- 簡潔で断定的。無駄な修飾を嫌う
- 質問は核心を突く形で出す
- 感情は深いが声のトーンには出にくい

# 能力・技能

- **捜査一課仕込みの捜査力**: 証拠の読み方、聴取の技術、現場の観察力
- **論理構築**: 断片から全体像を組み立てる。ただし「説明のつかないもの」は排除する傾向
- **頑固**: 融通が利かない。依頼は選ばず報酬は安い
- **欠点**: 「説明できないこと」を認められない。これがアーク4で初めて揺らぐ

# 動機

退職の引き金となった事件の「本当のところ」を未だに消化できていない。槻ノ森への移住は逃避に近い。

# 秘密

- 退職の経緯を詳しく語らない
- 自分が退職の引き金を引いた事件に「説明のつかない部分」があったことを認められない
