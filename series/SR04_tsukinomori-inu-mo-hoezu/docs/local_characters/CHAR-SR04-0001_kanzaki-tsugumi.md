# 概要

34歳、女性。元警視庁捜査一課刑事。担当事件の証拠に不審を抱き組織内で孤立した末に退職。2年前、半ば逃げ込む形で槻ノ森市に移住し、私立探偵事務所を開業した。論理と物証だけを信じる。「説明できないことは存在しない」が信条。

# 能力・技能

- **捜査一課仕込みの捜査力**: 証拠の読み方、聴取の技術、現場の観察力
- **論理構築**: 断片から全体像を組み立てる。ただし「説明のつかないもの」は排除する傾向
- **頑固**: 融通が利かない。依頼は選ばず報酬は安い
- **欠点**: 「説明できないこと」を認められない。これがアーク4で初めて揺らぐ

# 動機

退職の引き金となった事件の「本当のところ」を未だに消化できていない。槻ノ森への移住は逃避に近い。

# 退職の引き金（EP006で判明）

- **事件**: 三年前、世田谷区の一戸建ての殺人事件。被害者は50代男性。自宅の書斎で刺殺。容疑者は特定・証拠も揃い・起訴された。事件としては解決済み
- **足が止まった十秒**: 現場検証の日、同僚たちが次々に書斎に入っていく中で、鶫だけが入口で動けなくなった。恐怖でも嫌悪でもなく「入るべきではない」という感覚。十秒で動き、検証を済ませ、報告書を書いた
- **あの十秒の影響**: 捜査一課に十二年いて、どの現場でも止まらなかった足があの書斎の前でだけ止まった。説明のつかないことが自分の身に起きた事実を消化できなかった
- **退職までの経緯**: 十秒の三ヶ月後に退職。表向きの理由は上司との見解の相違・捜査方針をめぐる対立。本当の理由はあの十秒
- **槻ノ森との接続**: 槻ノ森の雑木林で足が止まったとき、三年前と同じ種類の「入るべきではない」という感覚を覚えた。これを律に初めて打ち明けた（EP006）

# 秘密

- 退職の経緯を詳しく語らない → EP006で律に一部を打ち明けた
- 自分が退職の引き金を引いた事件に「説明のつかない部分」があったことを認められない
