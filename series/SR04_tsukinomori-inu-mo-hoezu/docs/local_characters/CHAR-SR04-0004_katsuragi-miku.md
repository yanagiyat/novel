# 概要

21歳、女性。槻ノ森大学民俗学専攻3年。槻ノ森市生まれ育ち。地元の怪談・都市伝説・旧家伝承の収集を卒論テーマにしている。律の情報誌への読者投稿がきっかけで知り合い、非公式にアルバイトとして雇われた。

# 人格

- 明るくフラット
- 怖い話を怖がらずに記録する
- 「信じるかどうかより、あったかどうかが大事」が口癖

# 物語上の機能

異常ノイズの運び屋。彼女が語る伝承・怪談の断片が、鶫には「迷信」として、律には「気になる何か」として受け取られる非対称さを演出する。アーク2以降、彼女の収集記録が「失踪者リスト」と微妙に重なり始める。本人は「偶然の一致」だと思っている。

# 対人関係

- **鶫（CHAR-SR04-0001）**: 資料提供（鶫は迷信と見なしつつ参照する）
