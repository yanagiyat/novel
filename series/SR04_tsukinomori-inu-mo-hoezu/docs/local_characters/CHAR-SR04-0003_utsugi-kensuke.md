# 概要

47歳、男性。槻ノ森署刑事課係長。元警視庁勤務で鶫の先輩刑事。鶫が退職した経緯を半分だけ知っている。異動で槻ノ森署に来て5年。温厚だが保守的。「ここの事件は都内と違う。無理に掘るな」が口癖で、脅しでも懐柔でもなく本心からの忠告として言っている。

# 人格

- 温厚だが保守的
- 忠告は本心から
- 「無理に掘るな」は自分自身への戒めでもある

# 秘密

- 5年前に「掘りすぎた案件」の記憶が薄くなっていることに気づいている
- 気づいていて、掘り返さないことにした

# 物語上の機能

- 鶫の行動に対する公的な摩擦
- 「この街の長い住人は何かを知っている」と読者に感じさせるバロメーター役
- アーク3以降、彼の「忘れている何か」が伏線として機能し始める
