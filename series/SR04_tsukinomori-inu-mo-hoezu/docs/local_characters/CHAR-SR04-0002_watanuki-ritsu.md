# 概要

27歳、男性。地域情報誌「ツキノ森タイムス」の編集兼ライター、兼非公式の情報売り。表向きはひょうきんで人懐っこく、街の人脈が広い。「証明したいとも、証明したくないとも思っている」という曖昧な立場を自覚している。祖父が市内の旧家出身で、「この街には声が届かない場所がある」という言葉を子供の頃から聞かされてきた。

# 外見

（執筆時に確定。親しみやすい外見。取材道具を常に持ち歩いている雰囲気）

# 口調

- 一人称：「俺」
- 軽口が多い。だが核心に触れるときだけトーンが変わる
- 比喩や言い回しを好む（パターン5の要素）
- 情報を出すタイミングを計算している

# 能力・技能

- **街の人脈**: 広く浅い繋がりを多数持つ。表の取材ルートと裏の情報売りルートの使い分け
- **祖父のメモ**: 断片的な日記と走り書き。街の謎への手がかり
  - 祖父は市内の旧家出身。死後の遺品整理で走り書きのメモが複数枚出てきた
  - ほとんどは断片的で判読困難だが、はっきり読めるメモが複数枚ある:
    > **1枚目**:「静かに去る者を、追ってはならない」
    > **2枚目**（EP004で発見）:「森の奥に入ろうとする者は、足が止まる。それでいい。止まらなかった者が消える」
    > **3枚目**（EP005で発見）:「森の声は秋に近くなる。十月の半ばから。犬は先に聞く。」——犬の沈黙（10月10日発生）と時期が符合。万年筆のインクが少し滲んでいる
  - 祖父は生前「この街には声が届かない場所がある」と繰り返していた（EP001～）
- **直感**: 論理では説明できないが「そういうことだ」と感じる力。鶫とは対照的
- **欠点**: 曖昧な立場にいること自体が彼の安全弁であり、踏み込みきれない

# 動機

祖父のメモを整理するうちに街の謎に引き込まれていく。「この街には声が届かない場所がある」という言葉の意味を知りたい。

# 秘密

- 祖父のメモの全容をまだ把握していない
- 情報売りとしてグレーな仕事もしている

# 対人関係

- **神崎鶫（CHAR-SR04-0001）**: 邪魔にしつつ引き合う。認識のズレが緊張を生む
- **葛城実玖（CHAR-SR04-0004）**: 雇用と信頼。彼女の収集記録が鍵になる
- **蛇口良平（CHAR-SR04-0005）**: 旧知。利用し合う関係

# キャラクターアーク

1. **初期（ARC1）**: 鶫に何度も絡む邪魔な存在として登場
2. **中期（ARC2-3）**: 祖父のメモの整理を通じて街の謎に本格的に向き合う
3. **転換（ARC4）**: 鶫が揺らぐのと逆に「やっぱりそういうことだ」と落ち着いていく
4. **最終（ARC5）**: 祖父のメモの最後のページを見つける。新しい記事を書き始める。確信は持てないまま

# 登場履歴

- **SR04-EP004**: 祖父のメモ2枚目を発見し、鶫に電話で伝える。「森の奥に入ろうとする者は、足が止まる。それでいい。止まらなかった者が消える」
- **SR04-EP005**: 事務所に肉まんを差し入れに来訪。祖父のメモ3枚目を提示。低周波・超音波の仮説を提案するが鶫に否定される。「俺はもっと古いものだと思ってる」と、鶫との認識のズレを明言。鶫に「足が止まったとき、怖かった？」と核心を突く質問
