# 異常系統設定 — SR04

## 使用する異常系統

| P25ML系統ID | 名前 | 本作での役割 |
|-------------|------|-------------|
| 7 | 怪異（妖怪・心霊・都市伝説の実体化） | 槻ノ森の「静穏」を維持するなんらかの仕組みの実体。直接描かず「見た人間がどうなったか」の痕跡として登場 |
| 8 | 異界（現実と重なる別の層） | 西雑木林の奥に存在する可能性が示唆される空間。物語を通じて確定されない |

## 異常ノイズの設計

### 出現頻度

| アーク | 頻度 |
|--------|------|
| ARC1（1-20話） | 5話に1回 |
| ARC2（21-40話） | 3話に1回 |
| ARC3（41-60話） | 2話に1回 |
| ARC4（61-80話） | 毎話薄く |
| ARC5（81-100話） | 毎話・深く |

### ノイズの具体例

- 説明はつくがちょっと変な出来事（怪音・記録が残らない電話）
- 雑木林の奥に入ったまま戻った人間が翌日何かを忘れている
- 失踪者全員が消える直前に「変なものを見た」と周囲に漏らしていた
- 鶫自身が「説明のつかない体験」をする夜（ARC4）

### 描写原則

- 直接描写しない。痕跡・結果・証言としてのみ登場
- 小謎は必ず人間側の事情で解決する。超常で解決しない
- 読者が「偶然か異常か」を判断できない余白を維持する

## 静穏の正体

槻ノ森市が「異常が波及しにくい静穏地帯」である理由の正体。物語を通じて断片が集まるが、最終的に確定はしない。「見てしまった者はなんらかの形で排除される」が、登場人物たちはそれを直接知らない。感じるのはただ「この街はちょっとおかしいくらい静かだ」という違和感のみ。
