# SR04: 槻ノ森、犬も吠えず

## ジャンル
ミステリー・謎解き / 群像劇 / クライム

## コンセプト
「異常が波及しにくい静穏地帯」として知られる槻ノ森市で、元刑事の私立探偵・神崎鶫と地元の情報屋ライター・綿貫律が、互いに邪魔にしながら同じ事件を追い続ける。街では「一度解決した事件の関係者」が数年後に静かに消える。各話の小謎は人間の事情で解決されるが、解決の後に「なぜここでこれが起きたのか」という余白が残り、大きな謎の輪郭が浮かぶ。

## 文体
パターン2（ハード＆ロジカル文体）ベース。律の視点章のみパターン5（饒舌・言葉遊び）を薄く混ぜる。

## 核心ルール
「この街では、見た者は消える。でも殺されるわけではない」
- 鶫が「組織的な証拠隠滅」と解釈し、律が「そういう話じゃないと思う」と感じる認識のズレが全編の緊張の核

## 主要キャラクター

- **神崎 鶫**（34歳・女）: 私立探偵（元警視庁捜査一課刑事）。論理と物証だけを信じる。「説明できないことは存在しない」
- **綿貫 律**（27歳・男）: 地域情報誌ライター兼非公式情報売り。街の人脈が広く、祖父の残した謎を追う
- **宇津木 謙介**（47歳・男）: 槻ノ森署刑事課係長。元警視庁で鶫の先輩。「ここの事件は都内と違う」
- **葛城 実玖**（21歳・女）: 槻ノ森大学民俗学専攻3年。律のリサーチアルバイト。異常ノイズの運び屋
- **蛇口 良平**（38歳・男）: ジャグチ質店店主。グレーゾーンの情報ハブ。オブシディアン末端と接触
- **槻原 登貴子**（67歳・女）: 槻原家現当主。アーク3以降のキーパーソン。管理者であり黒幕

## 執筆時の注意

- 小謎は必ず人間側の事情で解決する。超常で解決しない
- 異常ノイズは痕跡・結果として描く。直接描写しない
- 槻原家を単純な悪として描かない。「管理者」であり悪人ではない
- 鶫と律の認識のズレを物語の推進力にする
- 街の住民が「見ないこと」を選んでいるのは洗脳でも陰謀でもなく、正気を守るための自然な選択
- 根本の謎は最後まで解決しない。人間側の黒幕のみ解決
