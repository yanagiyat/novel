# アーク概要

鶫が槻ノ森で事務所を開いたばかり。依頼はしょぼい。律が何度も邪魔をしてくる。街の人間関係と地理を読者に馴染ませながら、1話完結の小謎を20本積む。20話ラストで「解決した依頼人が後日静かに引っ越していた」というパターンが初めて可視化される。二人はまだ繋がりに気づいていない。

**異常ノイズ頻度**: 5話に1回
**小謎**: 毎話完結

# キャラクターの変化

- **鶫**: 開業→日常の依頼をこなす→微かな違和感を溜め始める
- **律**: 邪魔な存在→街の裏事情に詳しい人間として認識される
- **宇津木**: 温厚な先輩→「掘るな」と言う人間として印象づけ
- **実玖**: 明るいリサーチャー。伝承と現実の接点を運ぶ
- **蛇口**: 便利だが胡散臭い。何かを知っているようで知らない
