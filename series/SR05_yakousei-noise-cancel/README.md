# SR05: 夜行性のノイズキャンセル

## ジャンル
アーバン・サスペンス / サイコスリラー / 現代ファンタジー

## コンセプト
槻ノ森市の高校生・槙透は、SNSとAIが引き起こす「情報災害（認知汚染）」を強烈な耳鳴りとして感知する。論理と物理法則を武器にする男・飛羽に拾われ、都市を覆い尽くす「快適なファシズム」の波に、屁理屈とアナログな罠で立ち向かう。思考を放棄した世界で「間違える自由」を取り戻すための100日間。

## 文体
パターン3（シニカル＆ドライ文体）。疾走感のあるテンポ。バトルは魔法ではなく物理トリック＋情報戦。

## 核心ルール：非対称なハッキングバトル
- 敵：AIが計算した完璧な群集心理 + デジタルデータ
- 味方：不確定なアナログのノイズ
- 洗脳解除は「説得」ではなく【文脈の破壊（物理的エラー）】でAIの予測式をオーバーフローさせる
