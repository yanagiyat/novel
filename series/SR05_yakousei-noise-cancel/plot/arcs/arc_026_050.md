# アーク概要

「美しく安全な街づくり」が加速し、市民が自警団化する。調律師（チューナー）が本格稼働し、情報災害は「カルト的ブーム」から「完全同調」へ段階を上げる。飛羽は敵の罠にかかり重傷を負い退場。透は「テロリスト」のレッテルを貼られ、市民全体から追われる孤独な逃亡者となる。ハルカだけが辛うじて味方として残る。

**トーン**: 絶望×逃亡。味方を失い包囲される恐怖
**情報災害段階**: カルト的ブーム → 完全同調
