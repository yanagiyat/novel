# アーク概要

高校2年の槙透は事なかれ主義で目立たない日々を送っていた。ある日、裏バイト（落書き消し）の現場で激しい耳鳴りに襲われ、群衆が一斉に同じ方向を向く異様な光景を目撃する。謎の男・飛羽に拾われ、それが「情報災害（認知汚染）」であること、自分が認知汚染を感知する体質——炭鉱のカナリア——であることを知る。飛羽の指導のもと、局所的な情報災害を一件ずつ物理的に鎮圧していく。ハルカと出会い、「夜の住人たち」の世界に馴染んでいく。25話で「美しく安全な街づくり」キャンペーンが始動し、情報災害が市政レベルに拡大する兆しを見せる。

**トーン**: 導入×巻き込まれ。日常の裏側が徐々にめくれる
**情報災害段階**: 局所的炎上
