# 概要

旧市街の路地裏にある古い喫茶店。飛羽が常駐し、夜漏守の非公式な連絡拠点として機能している。店主は事情を知らないが、深夜営業を黙認している。ジャズのレコードが常にかかっており、店名の由来でもある。昼間は普通の喫茶店だが、深夜帯は「夜の住人たち」の溜まり場になる。
