# 概要

槻ノ森大学のキャンパス地下に残る旧共同溝。大学の再開発計画から取り残された区画で、公式には立入禁止。監視カメラの死角であり、夜の住人たちが情報交換や緊急避難に使う。パート3以降、透たちのゲリラ作戦の作戦司令部となる。
