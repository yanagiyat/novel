# 概要

エンリング（ORG-G-0002）が槻ノ森市に設置した大規模データセンター。調律師（チューナー）の中枢サーバーが物理的に格納されている。サヤカが派遣社員として勤務しており、内部構造を把握している。厳重なセキュリティと未解明技術（P25ML系統12）による防壁が施されている。最終決戦の主要舞台の一つ。
