# 概要

38歳、男性。非公式互助組織「夜漏守（よもすがら）」のトラブルシューター。喫茶「ブルー・ノート」に常駐。魔法や異能を憎悪し、「物理法則」と「行動経済学」だけで敵の異常性を地面に引きずり下ろす。

# 通り名

**引力（グラビティ）** — どんな非現実的な現象も物理法則の地面に引きずり下ろすことから。

# 秘密

**エンリング（ORG-G-0002）の初期開発者**。情報災害（AI）の基礎理論を作ってしまった張本人。自らの手で生み出した「システムに操られる思考停止の化け物（群衆）」を破壊するために戦っている。

# 戦術

- **物理法則**: あらゆる問題を物理的に解決しようとする。ゴムボール、音波、光学的錯覚など
- **行動経済学**: 群衆心理の予測と操作。AIの予測を逆手に取る
- **異能否定**: 魔法的な解決を拒否する。「説明できないなら壊せるはずだ」
- **文脈の破壊**: AIが予測できない「ナンセンスな物理的介入」で洗脳をオーバーフローさせる

# 動機

自らの手で生み出してしまった「システムに操られる思考停止の化け物（群衆）」の破壊。贖罪と破壊衝動が同居している。

# 対人関係

- **透（CHAR-SR05-0001）**: 拾った少年。カナリアとして使うが、徐々に弟子として認める
- **サヤカ（CHAR-SR05-0003）**: 古い協力者。飛羽の過去を部分的に知っている

# キャラクターアーク

1. **起（EP1-25）**: 透を拾い、局地的な情報災害を物理的に解決していく
2. **承（EP26-50）**: 敵の罠で重傷。退場
3. **転（EP51-75）**: 意識を取り戻し透と再会。過去（エンリング開発者）が暴露される
4. **結（EP76-100）**: 透と共に最終反撃。自分の作ったシステムに決着をつける
