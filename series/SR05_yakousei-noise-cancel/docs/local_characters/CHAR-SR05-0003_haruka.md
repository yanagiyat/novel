# 概要

22歳、女性。昼はエンリング（ORG-G-0002）のデータセンターで働く下請けの派遣社員。夜は飛羽たちの情報屋。ネット上のあらゆる監視網から自身のログを数分間だけ完全に消去する凄腕のクラッカー。いつも棒付きキャンディを舐めており、透をからかうのが趣味。

# 通り名

**空白（ブランク）** — 監視網から自身の存在を数分間消せることから。

# 能力・技能

- **ログ消去**: 監視網から自身のログを数分間完全消去。技術的スキルであり超能力ではない
- **内部情報**: エンリングの派遣社員としてシステムの内側にアクセスできる
- **制限**: 数分間のみ。大規模なハッキングには使えない。あくまで自分の痕跡を消すだけ

# 人格

- 軽口が多い。飄々としている
- 透をからかうのが趣味
- だが状況が切迫すると的確に動く
- 棒付きキャンディが常にある

# 対人関係

- **透（CHAR-SR05-0001）**: からかい対象。だがパート3以降は最も頼れる味方
- **飛羽（CHAR-SR05-0002）**: 古い協力者。飛羽の過去を部分的に知っている

# キャラクターアーク

1. **起（EP1-25）**: 情報屋として透に接触。エンリング内部の異変を察知
2. **承（EP26-50）**: 飛羽退場後、透の唯一の味方として機能
3. **転（EP51-75）**: 透のゲリラ戦を支援。自分のポジションの危うさが増す
4. **結（EP76-100）**: 最終決戦で物理回線ハッキングの要。二重生活の終わりを選ぶ

# 権限剥奪計画（EP006で構想）

- パラメータを注入している人間のアカウント権限を特定できれば、エンリングの内部システムを使って権限を直接剥奪できる可能性がある
- ハルカの昼の仕事（エンリング派遣社員）の範囲で「ぎりぎりできるかもしれない」
- **リスク**: バレたらクビ。ハルカ本人は「まあ、派遣だし。次の仕事探せばいいし」と軽口を叩くが、目は笑っていない
- EP006時点ではまだ権限の特定段階。実行には至っていない

# 登場履歴

- **SR05-EP004**: 初登場。ブルー・ノートのカウンターでノートパソコンを開いてエンリングのアクティビティログを追っている姿で登場。透に「カナリアくん？」と声をかける。三人の役割分担を「飛羽さんが腕、私が目、あんたが鼻。三人で一匹の犬」と表現。霞崎区の異常波形を検出し、槻ノ森市内での情報災害発生を予測。透を駅前カフェからリアルタイムでログ監視しながらナビゲートした
