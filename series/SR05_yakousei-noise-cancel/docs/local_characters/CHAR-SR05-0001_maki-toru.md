# 概要

17歳、男子。槻ノ森市立高校2年生。事なかれ主義で目立たないよう生きてきた。空間に「情報災害（認知汚染）」が充満すると、脳がドーパミンを拒絶し、激しい耳鳴りと頭痛を覚える体質を持つ（P25ML分類：超能力・系統9）。本人はただ苦痛なだけであり、洗脳されない以外の力は一切ない。

# 通り名

**炭鉱のカナリア** — 認知汚染の接近を耳鳴りで感知できることから。飛羽が命名。

# 能力

- **認知汚染感知**: 情報災害が充満すると激しい耳鳴りと頭痛。感知のみで戦闘力はゼロ
- **洗脳耐性**: 認知汚染に対して自動的に拒絶反応が出るため、洗脳されない
- **代償**: 情報災害が激化すると血を吐くほどの身体負荷
- **制限**: 感知だけ。解除もできず、他人を守ることもできない。ただの警報器

# 動機

「みんなが同じ方向を向いて笑っている時の気味の悪さ」への生理的嫌悪。

# 対人関係

- **飛羽（CHAR-SR05-0002）**: 師匠。拾われた恩人。後に彼の過去を知り、罪と覚悟を受け入れる
- **ハルカ（CHAR-SR05-0003）**: からかわれる後輩ポジション。だが実戦では頼りにしている

# キャラクターアーク

1. **起（EP1-25）**: 裏バイトで認知汚染に触れ、飛羽に拾われる。夜の住人としての生活開始
2. **承（EP26-50）**: テロリストとして追われる孤独な逃亡者に。飛羽を失う
3. **転（EP51-75）**: 飛羽の過去を知る。師匠の罪と覚悟を受け入れ、反撃を立案
4. **結（EP76-100）**: 「間違える自由」を突きつけ、名もなき日常へ帰る
