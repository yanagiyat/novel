# シリーズテンプレート

このディレクトリはP25MLの新規シリーズを作成する際のテンプレートである。
新しいシリーズを開始する場合、このディレクトリをコピーして `series/SR{nn}_{slug}/` にリネームする。

## 本文生成時の原則

### 文体と視点
- 文体・視点・テンポ・会話比率はシリーズ設定（`series.yaml`）に従う
- デフォルトは日本語の標準的な商業小説寄りの文体
- ただし各シリーズで自由に変更可能

### 品質方針
- 説明過多を避ける
- 裏設定を出しすぎて、単独シリーズの読み味を壊さない
- 世界観で読ませるが、キャラクターの魅力も軽視しない
- 日常劇と超常劇を同じアーキテクチャに載せられるようにする

### 裏設定の露出制御
- `world_exposure` の値に従い、裏設定の表出を制御する
  - `none`: 裏設定はほぼ出さない。一般小説として成立させる
  - `low`: 背景ノイズ程度に留める。不思議なニュース、説明のつかない小さな出来事
  - `medium`: 裏設定が物語を動かす。ただし世界の全貌は見せない
  - `high`: 裏設定が主題。世界の深層に直接触れる

### 共有世界の扱い
- 読者には共有世界であることを強く説明しない
- 自然に気づく構造を目指す
- 各シリーズは単独でも読めるように設計する

## ディレクトリ構造

```
series/SRXX_slug/
├── series.yaml          # シリーズ基本設定
├── README.md            # この説明ファイル
├── docs/                # シリーズローカルの設定
│   ├── local_setting/
│   ├── local_characters/
│   ├── local_organizations/
│   ├── local_locations/
│   └── local_items/
├── plot/                # プロット管理
│   ├── arcs/
│   ├── foreshadows/
│   ├── character_arcs/
│   └── open_questions.md
├── episodes/            # エピソード本文
├── timeline.md          # シリーズローカルタイムライン
├── change_log.md        # 変更履歴
└── status.md            # 進行状況
```
