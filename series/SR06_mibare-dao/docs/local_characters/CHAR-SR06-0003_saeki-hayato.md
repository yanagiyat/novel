# 概要

17歳、高校2年生の男子。直人の親友。柔道部所属で明るく気が利く、表向きは普通の親友。しかし古い家系に由来する護身・追跡・潜入・身体操作の技術を持っている。本人はそれを「変わった家庭のしつけ」程度にしか認識していない。

# 口調

- 一人称：「俺」
- 明るく素直
- 人との距離感がうまい
- 深刻な場面でも空気を和らげる力がある

# 能力・技能

- **柔道**: 部活動レベルの実力
- **護身術**: 家系由来。柔道とは別系統の実戦的な技術
- **追跡・潜入**: 尾行、侵入、気配の消し方
- **身体操作**: 常人離れした反射神経と運動能力
- **観察眼**: 人の動きや場の空気を読む鋭さ

# 匂わせ方

- 序盤は観察眼の鋭さや身体能力の高さを「柔道部だから」で自然に説明する
- 終盤で「え、お前そっち強いのか」が成立するように伏線を仕込む

# 物語上の役割

- 日常側の温度を支える
- 直人を「裏の住人」にしきらない防波堤
- 後半で護衛・潜入・脱出の場面で活躍する
