# 概要

29歳前後の女性。ダークサイドの危機管理コンサルタントにしてフィクサー。表に出せない案件の軟着陸を請け負い、暴力が出る前に順番を組み替える。交渉、回収、線引き、撤退、後始末が本領。

# 外見（EP004で確定）

- 二十代後半から三十歳前後
- 黒髪のボブカット
- グレーのコートにダークネイビーのパンツ。ヒールは低い（歩きやすさ優先）
- 革の鞄を肩にかけている
- 「意図的に印象を限定させない服装」——ビジネスカジュアルとも私服とも取れる

# 口調

- 一人称：「私」or「あたし」
- 実務家の口調。正論より順番を見る
- 容赦はないが、口が悪すぎるわけでもない
- 感情論を嫌い、状況の優先順位で語る

# 能力・技能

- **交渉・調整**: 利害関係者の面子と実利を読み、軟着陸の段取りを組む
- **回収・線引き**: 損切りと撤退ラインの見極め
- **後始末**: 事後処理と痕跡管理
- **追跡の剥がし方**: 尾行・監視への対処法
- **契約の読み方**: 法的・裏社会的な契約構造の把握

# 直人を助ける理由

善意100%ではない。複合的な動機で動く。

- 軟着陸させれば自分も儲かる
- 直人の才能に価値を見ている
- 放置すると面倒が拡大する

# 典型的な役割

- 死なない順番を教える
- 追跡の剥がし方を教える
- 契約の見方を教える
- 面子と撤退ラインの読み方を教える

## 「死なない順番」の具体的教え（EP006で開示）

| 順番 | 内容 | 意味 |
|------|------|------|
| 一歩目 | **動く前に2秒考える**。2秒で「この行動で誰に何が見えるか」を考える癖をつける | 善意の即応が自分の正体を晒すことを防ぐ |
| 二歩目 | **コンビニに入れ**。コンビニ＝監視カメラ＝素人が手を出しにくい場所。ドリンクを1本買って10分待って出る | 「安全な場所」はないが「相手が手を出しにくい場所」はある |
| 三歩目 | **小物を泳がせて中物を観察する**。小物を追い払うな。小物の動き（誰に報告するか、何を確認しに来るか）で中物の存在が分かる | 直接戦わず情報を集める。面倒くささを武器にする |

### 抑止の原則
- 直接戦わなくていい。相手に「面倒そうだ」と思わせればいい
- 大人のバックがいるように見せるだけで次のアクションが遅くなる
- 今の直人のレベルで使える最良の武器は「面倒くささ」
