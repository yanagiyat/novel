# SR06 執筆ルール

## トーン
現代ラノベ寄り。軽妙でエッジが効いた会話。難しすぎる専門用語で読者を振り落とさない。DAO・ブロックチェーン・暗号技術などは「何ができるか」で語り、「どう実装されているか」には深入りしない。

## 各話の基本構造
1. **日常導入** — 学校・家庭・友人との場面から入る
2. **裏側の進展** — Relay関連の事態が一歩動く
3. **会話の切れ味・判断ミス・次回への引き** — 読者が「次はどうなる」と思う引きで閉じる

## 各話で意識すること
- **情報を1つ進める** — 読者が知らなかったことを1つ開示する
- **関係性を1つ動かす** — キャラクター間の距離・信頼・緊張が1つ変化する
- **最後に不安か期待を1つ残す** — 読後感として「気になる」を置く

## 会話の軸となる三人
- **直人**: 賢いが青い。正しい判断ができるが、正しさが通用しない世界に直面する。
- **燈子**: 実務家。順番重視。感情ではなく手順で物事を動かす。商売人の視点を持つ。
- **隼人**: 明るく素直。体が強い。観察眼があるが、本人はそれを特別だと思っていない。

## 禁止事項
- **超能力バトルにしない** — あくまで現代社会の延長線上の物語
- **俺TUEEEにしない** — 主人公は「最善手を打つが悪化する」構造。賢さは武器だが万能ではない
- **エロ不要** — ラブコメ要素は空気として薄く存在する程度
- **魔法化しない** — 技術をご都合主義的な万能ツールにしない

## 暴力の温度
基本は知能戦・交渉戦。たまに生々しい暴力が侵食してくる程度。暴力が出たとき、それが「日常を壊す異物」として機能するように温度差を保つ。

## ラブコメ温度
薄め。師弟関係・信頼の蓄積が主軸。空気として少しだけ火花がある程度。恋愛で話を駆動しない。

## 日常描写の重要性
家庭・友人・学校イベントは飾りではなく**防衛線**。直人にとって守るべきものであり、それが侵食されるから痛い。日常が丁寧に描かれていないと、日常が壊れたときのダメージが読者に伝わらない。

## 日報制度（EP006で導入）

燈子が直人に課した安否確認の仕組み。

| 項目 | 内容 |
|------|------|
| 頻度 | 毎日1回（放課後） |
| 手段 | メッセージ（メインのスマートフォン） |
| 選択肢 | 「異常なし」または「接触あり」の二択のみ |
| 目的 | 直人の安全確認＋異常の早期検知 |

- EP006で初回の「異常なし」を送信。燈子から3秒で「よろしい。」と返信
- シンプルな制度だが、毎日の報告義務自体が直人の日常に「裏の世界」を持ち込む装置として機能する
- 「接触あり」が初めて送られるタイミングが今後の緊張ポイント

## 技術描写
- リアリティを保つが説明過多にしない
- 面白さ優先。正確さのために物語を止めない
- ただし雑な魔法化はしない。「それっぽい嘘」ではなく「省略された本当」を目指す
