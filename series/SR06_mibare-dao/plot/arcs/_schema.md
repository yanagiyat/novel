# プロットアークスキーマ

10話単位でアークを管理する。

## ファイル名規則
`arc_{開始話数}_{終了話数}.md`
例: `arc_001_010.md`, `arc_011_020.md`

## 構造

```markdown
---
arc_id: ARC-SRXX-001
series_id: SRXX
title: ""
episodes: "001-010"
theme: ""
status: planned
---

# アーク概要
（このアークで何を描くか）

# 主要な展開
| EP | タイトル案 | 要約 | ステータス |
|----|-----------|------|-----------|
| 001 | | | planned |
| 002 | | | planned |
| ... | | | |

# このアークで回収する伏線
（foreshadow_registry から参照）

# このアークで張る伏線
（新たに設置する伏線）

# キャラクターの変化
（このアーク開始時と終了時のキャラクター状態）

# メモ
```
