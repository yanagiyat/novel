# 伏線管理スキーマ

## 伏線レジストリのエントリ構造

```markdown
### FS-SRXX-001: 伏線タイトル

- **設置**: EP番号 — 設置の描写
- **回収予定**: EP番号またはアーク番号
- **回収済み**: 未 / EP番号
- **種別**: 小伏線 / 中伏線 / 大伏線
- **関連**: キャラクターID、イベントID等
- **メモ**: 補足
```

## 種別定義
- **小伏線**: 1-3話以内に回収される小さな仕掛け
- **中伏線**: アーク内で回収される中規模の仕掛け
- **大伏線**: 複数アークにまたがる大きな謎や仕掛け
