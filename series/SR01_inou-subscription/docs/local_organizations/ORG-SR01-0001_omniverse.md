# 概要

オムニバース社（Omniverse Inc.）は、階層都市ネオ・トウキョウの全インフラを管理する超巨大プラットフォーム企業。通信、決済、医療、教育、行政、そして異能（SaaS）のすべてを一社で運営する。EULA（エンドユーザー使用許諾契約）が国家法を超越する超特区において、事実上の統治者。

# 表の顔

- ネオ・トウキョウの全住民にサービスを提供する「生活基盤プラットフォーム」
- ニューロ・リンクの製造・保守
- エーテル・システム（スマート・ナノダスト）の散布・管理
- SaaS（異能）プランの提供
- 電子決済・行政手続き・医療・教育の統合管理
- 「すべての人に力を」がスローガン

# 裏の顔

- 無料プランユーザーの脳をクラウド計算資源として無断借用（生体ボットネット化）
- ユーザーの寿命を搾取してシステムの演算能力を確保
- EULA支配により、規約違反＝社会的死を実現
- 最終目標：全ユーザーへの洗脳アップデートによる完全管理
- モデレーター部隊（利用規約執行官）による暴力的な規約執行
