# 概要

クロヤが首から下げている分厚い自作デバイス。ジャンクパーツから組み上げた物理キーボードと、脱獄済みニューロ・リンクへの直結インターフェースを内蔵する。クロヤの戦闘スタイルの要。

# 機能

- **物理キーボード入力**: AR上のソフトウェアスワイプ（遅延あり）に対し、電気信号による直接割り込みが可能。物理法則的に必ず先手を取れる
- **詐欺UI生成**: 脱獄OSと連携し、敵のAR視界に偽の警告・同意ボタン・エラーメッセージを割り込ませる
- **近接ポート接続ケーブル**: 敵の後頭部ポートに密着させるための物理コネクタ（3秒間の接触が必要）
- **ローカル演算**: オムニバース社のクラウドを経由しない、デバイス内完結の演算処理

# 制約

- 射程ゼロ。すべての操作に物理的な近接が必要
- 詐欺UIの割り込みも、敵のニューロ・リンクの通信範囲内（数メートル）でないと届かない
- ジャンクパーツ製のため故障しやすい。クロヤが常に修理・改良している
- 最終章でRootにより破壊される

# 危険性

- 脱獄デバイスの所持自体がEULA重大違反
- 発覚すればアカウント永久凍結＋物理排除の対象

# 所有履歴

- クロヤが独学で設計・製作
- 部品はアンダー・ラグの闇市場で調達

# 備考

- この作品における「主人公の剣」に相当するアイテム
- 「物理入力がソフトウェア入力に勝る」というルールの象徴
- 最終決戦で破壊された後、クロヤは自身の腕に直接端子を刺して代替する
