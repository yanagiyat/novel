# 概要

アンダー・ラグは、ネオ・トウキョウの下層に広がるスラム区画。無課金・低額ユーザーの居住区で、中継アンテナが劣化しており、常に数十ミリ秒のラグやパケットロスが発生する。飛行中のギガ死（通信制限）による墜落事故が多発する危険地帯。

# 表の構造

- 劣化した中継アンテナが点在
- ジャンクパーツを再利用した自作の建物・インフラ
- 闇市場（フリマアプリの物理版のようなもの）
- 違法改造ショップ、無許可クリニック
- 常にARノイズが走っており、視界が不安定

# 裏の構造

- 無料ユーザーの脳が巨大ボットネットの計算ノードとして利用されている
- スラムの住人は自分の寿命が削られていることをほとんど知らない
- 脱獄済みのデバイスを扱う非公式技術者が少数存在する
- 一部のエリアは通信が完全に途絶しており、ARすら機能しない「圏外地帯」がある

# 立入条件

- 上層からは「降りる」ことで到達可能だが、好んで来る上層民はいない
- 圏外地帯はモデレーターの監視も届かない

# 注意事項

- クロヤのホームグラウンド。日常シーンの主舞台
- 悲惨さだけでなく、スラムなりの活気やユーモアも描くこと
- 技術的に劣悪な環境こそがクロヤの戦術を鍛えたバックグラウンド
