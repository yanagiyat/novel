# 概要

アッパー・クラウドは、ネオ・トウキョウの上層に位置する富裕層・高額課金者の居住区。巨大エッジサーバーに物理的に近接しており、Ping値0.1ミリ秒以下のラグなし5G通信環境を誇る。AR演出は美しく安定し、広告は一切表示されない。リサの出身地。

## 五層構造（EP004でリサが証言）

| 層 | 名称 | 人口比 | 月額課金 |
|----|------|--------|----------|
| 1 | スタンダード | 60% | 9,800円 |
| 2 | プレミアム | 25% | 49,800円 |
| 3 | ビジネス | 10% | 198,000円 |
| 4 | エンタープライズ | 5%未満 | 非公開（契約ごと。年間八桁〜） |
| 5 | Root直轄階層 | 公式には存在しないことになっている | 不明 |

- Root直轄階層の住人は「テナント」と呼ばれる
- Rootの声を直接聞いたことがある人間はリサの知る限り3人のみ
- 指示はシステム経由で下る

# 表の構造

- 浮遊プラットフォーム上に建設された高級居住区
- 巨大エッジサーバー施設（通信の物理的中心）
- プレミアム・エンタープライズ契約者専用の商業施設
- オムニバース社の管理タワー
- AR演出が常に美しく、天候すらカスタマイズ可能

# 裏の構造

- エンタープライズ契約者は一般人には不可視のコマンドを使用可能
- 上層の豊かさは下層の生体ボットネットが支えている
- 権力闘争と陰謀が日常。リサはその犠牲者
- モデレーター本部がここにある

# 立入条件

- スタンダード以上のプランが必要
- 無課金ユーザーは物理的にもアクセスを阻まれる
- 第3〜4章でクロヤとリサが侵入する舞台

# 注意事項

- 豪華で美しいが空虚な世界として描く
- 課金社会の頂点にある「快適だが歪んだユートピア」
