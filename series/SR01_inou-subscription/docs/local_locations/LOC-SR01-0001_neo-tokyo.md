# 概要

階層都市ネオ・トウキョウは、オムニバース社が都市インフラ全体を管理する超特区。物理的に上層（アッパー・クラウド）と下層（アンダー・ラグ）に分かれた垂直構造を持ち、課金プランにより居住区・通信品質・社会的権利がすべて階層化されている。EULA（エンドユーザー使用許諾契約）が国家の憲法・刑法よりも優先される特異な法体系が敷かれている。

# 表の構造

- 全人類の後頭部に埋め込まれたニューロ・リンクにより、視界は常時AR化
- エーテル・システム（大気中のスマート・ナノダスト）がすべての異能の基盤
- 上層と下層の間は物理的なエレベーターシャフトと空中回廊で接続
- 商業・行政・医療・教育すべてがオムニバース社のプラットフォーム上で運営

# 裏の構造

- 無料ユーザーの脳内チップは余剰処理能力をオムニバース社に無断借用されている（生体ボットネット化）
- 寿命と引き換えに巨大なクラウド計算資源の一部にされている
- 都市の中心部にオムニバース社のデータセンター（物語第3章の舞台）がある
- EULA支配社会は他都市への拡張モデルとして設計されている

# 立入条件

- 都市内の移動は課金プランに応じて制限される
- 上層への移動にはスタンダード以上のプランが必要
- データセンターは最高機密区画

# 注意事項

- P25ML共有世界における時間軸・位置づけは保留（未来の東京、並行世界の東京、いずれも可能）
- 現実の東京の地名は使用可能だが、すべてネオ・トウキョウ内の地区名として再解釈する
