# 概要

第3章〜第4章の舞台。オムニバース社の全システムが格納された中央データセンター。物理的には都市の最深部に位置する。Rootの居城であり、SaaSシステムの中枢。全ユーザーの生体ボットネットの統括ノードもここにある。

# 表の構造

- 多層構造のサーバーファーム
- 各階層にファイアウォールに対応する物理的防壁
- エリアマネージャー（中ボス級モデレーター）が各層を管理
- 最深部にRoot専用の管理コンソール

# 裏の構造

- 利用規約（EULA）の条文が物理的な壁として実体化している
- 数万文字の規約の論理矛盾を見つけることで壁を突破できる
- 全ユーザーの洗脳アップデート配信装置
- Rootの権限は物理的にこのデータセンターに紐付いている

# 立入条件

- エンタープライズ以上の権限が必要（通常時）
- クロヤたちはDDoS攻撃の陽動で侵入する

# 注意事項

- 「RPGの最終ダンジョン」の構造をIT用語で再解釈した舞台
- 各階層がIT概念（ファイアウォール、EULA、サーバールーム等）に対応
- 最終決戦の舞台として、Rootの権限の「物理的な根拠」がここにある設定を活かす
