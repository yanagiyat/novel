# ニューロ・リンク

## 概要

全人類の後頭部に埋め込まれた生体チップ。視界を常時AR（拡張現実）化し、SaaS異能の発動端末として機能する。オムニバース社製。出生時に全員に埋め込まれ、取り外しは違法かつ技術的に極めて困難。

## 機能

- **AR視界**: 常時拡張現実表示。UI、ナビ、通信、決済、すべてがAR上で行われる
- **SaaS端末**: エーテル・システム（ナノダスト）への量子暗号通信の送信機
- **生体認証**: 個人識別。プランの紐付け。社会的IDそのもの
- **後頭部ポート**: 物理接続用のポート。クロヤの近接ハック戦術の対象

## 脱獄（ジェイルブレイク）

- 純正OSを改造オープンソースOSに書き換えること
- 違法行為。発覚するとアカウント凍結
- 広告表示を回避できるが、異能の威力は上がらない（ハードウェア性能は変わらないため）
- クロヤは脱獄済み。これにより詐欺UIの生成・表示が可能

## 暗黒面

- **生体ボットネット**: 無料プランユーザーの脳の余剰処理能力を無断借用
- 演算リソースとしてオムニバース社のクラウドに組み込まれている
- 結果として寿命が削られるが、ユーザーには通知されない
- この事実はFreeプランの利用規約の最深部（第847条の3）に極小フォントで記載されている
