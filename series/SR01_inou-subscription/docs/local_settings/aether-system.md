# エーテル・システム

## 概要

オムニバース社が大気中に散布した「環境制御型スマート・ナノダスト」。ネオ・トウキョウの大気中に遍在する目に見えないナノマシン群であり、SaaS異能の物理的基盤。

## 仕組み

1. ユーザーがニューロ・リンクから異能発動コマンドを送信
2. 量子暗号通信でエーテル（ナノダスト）に指令が届く
3. ナノダストが局所的に集合・組織化し、物理法則を一時的に書き換える
4. 書き換えの規模＝動員するナノダストの量＝必要な演算量＝課金プランの上限

## 制約

- ナノダストの密度はエリアによって異なる（上層は高密度、下層は低密度）
- 密度が低いエリアでは高負荷能力が発動しにくい
- 圏外地帯ではナノダストも枯渇しており、異能そのものが使えない
- ナノダスト自体はオムニバース社が定期的に補充散布している

## 物語的な重要性

- 異能の仕組みに「ナノマシン」という科学的（SF的）根拠を与える
- 課金プランによる威力差を「演算量と通信帯域の差」として合理化する
- 下層での戦闘不利を「ナノダスト密度の低さ」として物理的に説明できる
