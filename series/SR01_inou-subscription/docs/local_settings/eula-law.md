# EULA至上主義法体系

## 概要

ネオ・トウキョウは「エンドユーザー使用許諾契約（EULA）」が国家の憲法や刑法よりも優先される超特区である。オムニバース社の利用規約が事実上の法律であり、規約違反はアカウント凍結＝社会からの完全な抹殺を意味する。

## 規約の特徴

- **数万文字の超長文**: 誰も全文を読まない（読ませない設計）
- **頻繁な改訂**: ユーザーへの通知は極小バナーのみ
- **矛盾条項**: 膨大な条文の中に論理的な矛盾が存在する。クロヤはこれを突く
- **自動同意**: ニューロ・リンクの起動時に自動同意される
- **第847条の3**: 無料プランの生体ボットネット化条項（極小フォント）

## 規約違反の処罰

| 違反レベル | 処罰 |
|-----------|------|
| 軽微 | 一時的な能力制限 |
| 中程度 | プランの強制ダウングレード |
| 重大 | アカウント凍結（BAN） |
| 最重大 | アカウント永久凍結＋物理排除 |

## アカウント凍結（BAN）

- すべての異能が停止
- AR視界が「ACCOUNT SUSPENDED」で上書き
- 社会的サービス（決済、通信、移動、医療）へのアクセスが遮断
- 事実上の社会的死
- リサはこの状態にある
- **BAN者への物理的危険**: BAN者はスラムで妙な連中（闇組織、人体パーツ狙いの業者等）に絡まれるリスクが高い。社会的保護を一切失っているため、格好の標的になる

## 回収屋（債権回収部門）（EP006で判明）

オムニバース社の**債権管理部**に所属する合法的な取り立て要員。モデレーターが「暴力装置」であるのに対し、回収屋は「経理の暴力」——法的に、契約上、完全に合法な手段でアカウントを差し押さえ、デバイスを回収し、チップの使用権を取り消す。

| 項目 | 内容 |
|------|------|
| 所属 | オムニバース社 エンタープライズ債権管理部 |
| 外見 | 黒いスーツ。AR装甲ではなくスーツ。胸ポケットにオムニバース社のロゴ刺繍 |
| 声質 | 慇懃で冷たい。モデレーターの事務的な声よりさらに上品で、さらに非情 |
| 手段 | 書類と契約による合法的な差し押さえ。暴力は使わない |
| 主な対象 | 滞納者、凍結アカウント保有者 |

### 堂本（EP006で登場）
- エンタープライズの債権管理部所属
- リサの家のアカウント凍結手続きを執行した人物
- 闇市場に出現し、BAN者（リサ）を探索中
- スタンダードの末端とは格が違う上層の人間

## 物語的な重要性

- 規約の論理矛盾を見つけて壁を突破する＝「法の隙間を突く」戦術のSF化
- 第3章で利用規約が物理的な壁として実体化するギミックの基盤
- 「誰も読まない規約に人生を支配される」現実社会への風刺
