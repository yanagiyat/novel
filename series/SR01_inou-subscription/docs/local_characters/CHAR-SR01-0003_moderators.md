# 概要

オムニバース社の「利用規約執行部門」に所属するエリート戦闘員の総称。規約違反者の捕縛・アカウント凍結・場合によっては物理的排除を担当する。会社の経費で青天井に課金アイテムやSSRガチャ武器を乱用できる、Pay to Winの極致。

# 外見

- 統一されたスーツ型AR装甲（黒基調、オムニバース社のロゴ入り）
- ランクにより装備のグレードが異なる（低ランクはスタンダード相当、幹部はプレミアム以上）
- 全員がARサングラス型のヘッドアップディスプレイを装着

# 口調

- 事務的。規約条文を引用しながら攻撃する
- 「利用規約第XX条に基づき、貴殿のアカウントを凍結します」
- 上位になるほど慇懃無礼になる

# 能力・技能

- 会社経費による無制限課金で、プレミアム〜エンタープライズ級の異能を使用
- 戦闘中のリアルタイム決済で攻撃力バフ・自動回復を即座に適用
- SSRガチャ武器の乱用（物語的には「札束で殴る」戦闘スタイル）
- ただし、全員が**経費頼り**であり、個人の技量や応用力は低い
- クレジットカードの限度額を超過すると戦闘不能になるという弱点

# 対人関係

- Root（CHAR-SR01-0004）の命令で動く
- 内部にも階級があり、出世競争がある
- クロヤを当初は「ただの無課金スラム住民」として舐めている

# 禁止事項

- 全員を無個性な雑魚にしないこと。章ごとの中ボスクラスは個性を持たせる
- 「経費で強い」という設定を活かし、自力では弱いことを戦闘で見せる
