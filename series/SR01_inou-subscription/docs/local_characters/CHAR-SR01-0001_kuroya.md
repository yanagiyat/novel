# 概要

17歳の少年。階層都市ネオ・トウキョウのアンダー・ラグ（下層スラム）に住む無課金ユーザー。悪質な自動更新サブスクの罠で両親を破産・過労死させたオムニバース社への復讐を誓い、独学でニューロ・リンクの脱獄（ジェイルブレイク）に成功。課金ゼロのまま、詐欺UI・ソーシャルエンジニアリング・ゼロ距離物理ハックで課金エリートを打倒する。

# 外見

- パーカーのフード（常に被っている。顔の上半分が影になる）
- 目の下に深い隈（睡眠不足と栄養失調）
- 首から下げた分厚い自作物理キーボード付きデバイス（ITEM-SR01-0001）
- 痩身。身長は平均よりやや低い
- AR透かし「Omniverse Free」が常に体に表示されている（本人は気にしない）

# 口調

- 一人称：「俺」
- 冷めた口調。皮肉屋。感情をあまり出さない
- 戦闘時は極めて事務的。「はい、ここ押して。——お疲れ様でした」のような詐欺コールセンター的な冷酷さ
- IT用語を自然に会話に混ぜる
- 稀に両親のことが絡むと声が低くなる

# 能力・技能

## 戦闘スタイル：「課金額ゼロの頭脳戦」

**重要制約：遠隔ハッキング（魔法的チート）は一切できない**

1. **ソーシャル・エンジニアリング**: 敵のAR視界に偽の警告（サポート詐欺）や透明な同意ボタン（クリックジャッキング）を割り込ませ、敵自身に自滅のボタンを押させる
2. **ハードウェア割り込み（ゼロ距離物理ハック）**: 敵のAR視界でのスワイプ操作（ソフトウェア処理で遅延あり）に対し、物理キーボードの打鍵（電気信号による直接割り込み）で対抗。物理法則的に必ずクロヤの入力が勝る
3. **近接ポート接続**: 敵のシステムを書き換えるには、敵の後頭部にある生体ポートに物理デバイスを**3秒間**密着させる必要がある。命がけの接近戦（体術）が必須

## その他の技能

- 規約文書の精読と論理矛盾の発見
- スラム仕込みの喧嘩殺法（体術）

# 対人関係

- **リサ（CHAR-SR01-0002）**: パートナー。最初は利害関係の契約だが、互いに欠けた部分を補い合う。クロヤの戦術眼とリサの戦闘力の組み合わせ
- **両親**: 故人。悪質な自動更新サブスクの罠で破産、過労死した。クロヤの復讐の原動力
- **オムニバース社**: 復讐対象。しかし個人的な怨恨に留まらず、システムそのものへの怒り
