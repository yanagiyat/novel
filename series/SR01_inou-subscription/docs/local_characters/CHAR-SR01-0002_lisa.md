# 概要

17歳の少女。元アッパー・クラウドのエンタープライズ契約者。名家の令嬢だったが、陰謀によりアカウントを永久凍結（BAN）され、一切の異能と社会的権利を剥奪されてアンダー・ラグに追放された。クロヤと出会い、BAN解除のために契約を結ぶ。

# 外見

- 銀髪ロング（手入れが行き届かず、スラム生活で少し汚れている）
- 高価な軍服風ドレス（かつての上層生活の名残。汚れているが捨てない）
- 凛とした顔立ち。目つきが鋭い
- 凍結状態のため、AR表示が「ACCOUNT SUSPENDED」のウォーターマークで覆われている
- 身長はクロヤよりやや高い

# 口調

- 一人称：「私」（わたくし、ではなく「わたし」。育ちの良さは残るが堅すぎない）
- 丁寧だが芯が強い。怒ると言葉が短くなる
- サブスクや無課金の常識を知らず、天然ボケ的な発言をする
- 物語が進むにつれ、クロヤの影響でゲスい詐欺用語を覚えていく

# 能力・技能

## 戦闘能力
- 元エンタープライズ契約者として高度な異能を保有していた
- 凍結中は一切の異能が使用不可
- ただし、無料プラン相当の最低限の能力は残っている可能性がある
- **広告中の無敵時間**: 無料プランの5秒間広告が流れている間、入力を一切受け付けない＝攻撃も受け付けない。この仕様を逆手にとり、防御手段として応用する（第2章以降）

## その他
- 上層社会の知識（組織構造、人脈、政治）
- 高い戦闘センス（元エリート訓練済み）
- エンタープライズ級のセキュリティ知識
- 物語後半では、自ら詐欺UI戦術を立案するようになる

# 対人関係

- **クロヤ（CHAR-SR01-0001）**: 最初は利害関係の契約。クロヤの非道な戦術に嫌悪感を持つが、次第に「規約の穴を突く快感」に染まる。パートナー
- **オムニバース社**: 自分を陥れた陰謀の首謀者を探している
- **元上層の知人**: 凍結により関係断絶。一部は敵に回っている可能性

# 知っていること

- アッパー・クラウドの社会構造と権力関係
- エンタープライズプランの詳細仕様（一般人には不可視のコマンド等）
- 自身が陥れられた陰謀の断片的情報
- 上層のセキュリティ体制

# 知られていること

- 上層では「規約違反で凍結された令嬢」として処理されている
- スラムでは「やたら偉そうなBAN者」として浮いている
- モデレーターからは追撃対象

# 禁止事項

- 凍結解除を安易に行わないこと。物語の核心的な目標として温存
- 単なるお色気要員にしないこと。戦闘力と知性を持つヒロイン
- 堕落させすぎないこと。ゲスさを学びつつも正義感の芯は残す

# キャラクターアーク

1. **初期**: 上層の常識しか知らない世間知らず。無料プランの広告や透かし文字に絶望する
2. **中期**: クロヤの戦術を見て「下品だが合理的」と認め始める。広告無敵時間の防御応用を自力で発見
3. **後期**: 自らゲスい罠を立案するようになる。「この規約の23条7項、矛盾してるわ」
4. **最終**: 広告付きの氷の拳でRootを物理的に粉砕。無課金でも戦える証明
