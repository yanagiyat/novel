# 概要

オムニバース社の最高責任者にして、SaaSシステムの設計者。Sudo（絶対権限）コマンドを保有し、空間の書き換え、他者の能力の強制停止、さらにはあらゆる事象を「Delete」の一言で消去できる無敵の存在。全ユーザーへの洗脳アップデートを企てる最終敵。

# 外見

（詳細は物語進行に合わせて決定。以下は方向性）
- 清潔感のある中年男性、または性別不詳の印象
- 一切のAR装飾を必要としない（管理者権限で世界そのものが自分の端末）
- 極めて穏やかな外見。恐ろしさは態度と能力から伝わる

# 口調

- 穏やか、論理的、上から目線だが声を荒げない
- システム用語を日常会話のように使う
- 「Delete」「Permission denied」「Access granted」など、コマンドが言葉として発動する

# 能力・技能

- **Sudoコマンド**: システム管理者の絶対権限。あらゆるコマンドの認証をバイパスできる
- **Delete**: 対象を事象ごと消去
- **他者能力の強制停止**: 任意のユーザーのSaaSを無効化
- **空間書き換え**: 物理法則の局所的な改変
- **無限回復・無限課金**: システムの内側にいる限り、リソースに上限がない

## 弱点
- 絶対権限への過信（驕り）
- UIを通じた入力は、Root自身も例外ではない
- 自分が作ったシステムの仕様を「完璧」と信じている

# 対人関係

- モデレーター部隊の最高指揮者
- 全ユーザーを「自分のプラットフォームの利用者」としか見ていない
- クロヤの存在を最終章まで認識しない（無課金ユーザーは文字通り「見えない」）

# 知っていること

- SaaSシステムの全仕様
- 無料プランの暗黒面（生体ボットネット化）の設計者本人
- 全ユーザーの行動データ

# 禁止事項

- 序盤〜中盤で直接登場させないこと。存在感は「システムの圧」として間接的に伝える
- 単なる悪の独裁者にしない。Rootなりの「システム全体の最適化」という論理がある
- 最終決戦での敗北は「驕りから自滅のボタンを押す」形にする（クロヤの戦術の一貫性）

# EP004で判明した情報

- Root直轄階層は公式には存在しないことになっている（リサの証言）
- Rootと直接話した人間はほとんどおらず、声を聞いたことがある人間はリサの知る限り3人
- 指示はシステム経由で下る。一人なのか複数なのかも不明
- エンタープライズ契約の使途不明金「システム最適化負担金」（契約の15%、年間総額約200億）がRoot直轄階層と関連する可能性

# 登場履歴

- **SR01-EP004**: 直接登場なし。リサの証言を通じてRoot直轄階層の存在と、声を聞いた人間が3人しかいないことが明かされる
