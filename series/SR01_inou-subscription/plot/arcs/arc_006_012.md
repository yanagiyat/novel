# アーク概要

オムニバース社がスラム全体の掃討作戦を開始。クロヤとリサはスラムの住人と共にアンダー・ラグを防衛する。新戦術の開発と、リサの無料プラン適応が描かれる。

**このアークで読者に見せるもの**:
- 戦術の幅の拡大（ワンクリック詐欺、Cookie同意トラッキング等）
- リサの成長（広告無敵時間の発見）
- スラムの住人たちとの絆
- Freeプランの暗黒面のさらなる示唆

# キャラクターの変化

- **クロヤ**: 一人の復讐者→スラムの住人を守る立場に。まだ「利用しているだけ」と言い張る
- **リサ**: 無料プランへの絶望→逆手に取る発想の転換。クロヤの戦術を「下品だが合理的」と認め始める

# メモ

- リサの「広告無敵時間」発見は本作の象徴的シーン。デバフを逆手に取る発想
- エリアマネージャーは個性的なキャラにする。経費で湯水のように課金する「会社の金で遊ぶ嫌な上司」的キャラ
- スラムの住人たちにもキャラクターを持たせ、「守る理由」を読者に感じさせる
