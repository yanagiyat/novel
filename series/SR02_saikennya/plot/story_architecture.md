# 再建屋 ── 100話構造設計書

## 全体テーマ

**核心の問い**: 「組織は誰かの正義のために動けるか、それとも組織自体の維持のために動くのか」

**サブテーマ**:
- 壊された人間は、壊された方法以外の手段で再建できるか
- 正義と秩序は共存できるか。必要悪を認めた上で、何を変えられるか
- 信頼とは、相手を信じることではなく、相手の行動を記録し続けることか

## どんでん返し設計（5段階）

### 第1転換: EP019-020「来栖の影」
- **内容**: 詩音が機構のクラウドから取得した断片データの中に、来栖朔の名前がある。ただし「職員」としてではなく「クリーナー」として。来栖が壊した人間のリストの中に、詩音の元勤務先ITベンチャーの名前がある
- **衝撃**: 来栖は詩音の会社を潰す工作に関与していた可能性がある。詩音はまだ来栖と出会っていない（来栖合流はEP034以降）が、読者にはこの事実が先に提示される
- **仕込み**: EP014-016で詩音のITベンチャー壊滅の経緯を描いておく。EP020のデータで「KURUSU」の名前が一瞬映る
- **効果**: 第2部での来栖合流に対する読者の不信感を最大化する

### 第2転換: EP040「泳がされた勝利」
- **内容**: 第2部で再建屋が手がけた六人目（厚労省元係長）の救出が成功したかに見えた。だがEP040のラストで、奏が帳簿をつけている最中に気づく——再建に使った手段の一部が、機構側から「用意されていた」。六人目を助けるために使った情報ルートが、機構が意図的に開けた穴だった
- **衝撃**: 再建屋の最初の成功は、機構に「泳がされていた」。機構は再建屋の存在を把握し、わざと成功させて行動パターンを観察していた
- **仕込み**: EP035-039で来栖が提供する情報の「正確すぎる」部分を伏線として配置。桐嶋がEP040で裏金の帳簿に怒るのは、この構造に気づいたから
- **効果**: 来栖への疑惑が爆発する第3部への直接の導火線

### 第3転換: EP058-060「封筒の差出人」
- **内容**: 来栖が信用問題を解決するために単独行動を取る。その過程でリーク元が来栖ではなく、機構の内部改革派だったことが判明。さらに、改革派が奏に封筒を送った人物と同一であることを示す間接証拠が出る
- **衝撃**: 封筒の差出人は来栖ではなく、機構内部の改革派。ただし改革派の真の目的は「機構を壊すこと」ではなく「機構を作り直すこと」。奏たちは改革派にとっての「道具」でもある
- **仕込み**: EP001の封筒の精度、EP005で桐嶋と奏が立てた仮説「受取人の反応を見ている」をここで回収
- **補足**: ただしこの時点では「間接証拠」にとどめる。確定はEP095

### 第4転換: EP078-080「必要悪の証明」
- **内容**: 機構の評議部に近づく過程で、御手洗の30年分の取材資料と詩音のデータ解析が交差する。1990年代の金融危機、2011年の震災後の混乱、複数の政治危機において、機構が「調整」を行わなければ社会的パニックが起きていた証拠が出る
- **衝撃**: 機構を潰せば、これまで機構が抑えていた矛盾が一気に噴出する。正義のために戦ってきた再建屋が、正義を貫けば社会を壊す側になる
- **仕込み**: EP065の御手洗の旧コネクション情報、EP075の機構内部の割れ目、EP082での不都合な証拠と連動
- **効果**: 第5部で奏が「壊す/引き継ぐ/第三の道」の選択を迫られる基盤

### 第5転換: EP095-100「奏の選択」
- **内容**: 評議部の実態が判明する。評議部のメンバーは互いの顔すら知らない。機構は「誰かの意志」ではなく「仕組みそのもの」として自律的に動いている。奏は機構を壊すのではなく、機構の仕組みを「帳簿として可視化する」ことで無力化する道を選ぶ——暴力でも信用破壊でもなく、透明化
- **最終的な逆転**: 奏が同じ手（匿名で情報を操作する手法）を使いかけるが、桐嶋が「それは機構だ」と止める。チーム全員で別の方法を見つける
- **帳簿モチーフの回収**: 「帳簿は嘘をつかない」——機構の全活動を帳簿化し、公開可能な状態にすることが、最終的な武器になる
- **来栖の真意**: 来栖は最初から「自分が壊した人間たちの前に出る」ことを覚悟していた。封筒を送ったのは改革派だが、改革派に奏の名前を教えたのは来栖だった

## モチーフ設計

### 「帳簿は嘘をつかない」
| 段階 | EP | 用法 |
|------|-----|------|
| 設置 | 001 | 奏が封筒の資料を「帳簿を読むように」分析する |
| 発展 | 040 | 裏金を帳簿に記録する。帳簿で組織の倫理を可視化する |
| 危機 | 070 | 奏の感情が判断に影響し、帳簿的思考が揺らぐ |
| 試練 | 093 | 帳簿的手法で機構と同じことをしかける（匿名の情報操作） |
| 着地 | 096 | 機構の全活動を帳簿化・可視化することが最終武器になる |
| 回収 | 100 | 奏が新しい帳簿を開く。白紙のページに、最初の一行を書く |

### 「コーヒーの濃さ」
| 段階 | EP | 用法 |
|------|-----|------|
| 設置 | 001 | 規定量の半分。貧しさと習慣の境界が曖昧 |
| 日常 | 006 | 調査費が生活を圧迫。コーヒーの薄さは生活の余裕のバロメーター |
| 変化 | 026 | 旧社屋で御手洗がドリップコーヒーを淹れる。奏が少し戸惑う |
| 転機 | 049 | 作戦成功後、五人で飲むコーヒー。奏が初めて濃いコーヒーを飲む |
| 危機 | 070 | 一人になった奏が、また薄いコーヒーを淹れている |
| 着地 | 100 | エピローグ。規定量の粉をきっちり量る。薄くもなく濃くもなく、ちょうどいい一杯 |

### 「奏の表情は変わらなかった」
| 段階 | EP | 用法 |
|------|-----|------|
| 設置 | 001 | 封筒を読んでも表情は変わらない。変える必要がない |
| 反復 | 020, 040, 060 | 各部の節目で繰り返される。読者に「いつ変わるのか」を待たせる |
| 亀裂 | 070 | 「表情は変わらなかった。ただ、手が震えていた」 |
