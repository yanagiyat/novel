# アーク概要

五人が初めて本格的な作戦を実行し、調停機構の中間層にダメージを与える。来栖の信用問題が爆発し、組織が揺らぐ。

**リズム**: 作戦が進むごとに何かが変わっていく手触り
**柔軟区間**: EP51-60は来栖の信用問題の着地次第で展開を大きく変形できる

# キャラクターの変化

- **奏**: ルールの維持と仲間への信頼の間で揺れ始める
- **来栖**: 疑われ、自ら証明し、信用と傷が同時に残る

- **桐嶋と奏**: 台詞なしで関係が深まる
