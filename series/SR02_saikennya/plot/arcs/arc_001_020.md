# アーク概要

奏が封筒を受け取り、自分の人生が壊された構造を解明し始める。桐嶋、詩音と合流し、三人で調停機構の存在を証明する最初の一歩を踏み出す。

**リズム**: 謎がひとつ解けて新しい謎がひとつ生まれる

# このアークで張る伏線

- FS-SR02-001: 封筒の差出人の正体（誰が奏に資料を送ったのか）
- FS-SR02-002: 資料の「一箇所だけ省略されている」部分の意味
- FS-SR02-003: 桐嶋の娘の存在（EP13で匂う。ARC4で機構に接触される）
- FS-SR02-004: 詩音の壁一面の相関図（2年間の蓄積の価値）
- FS-SR02-005: 御手洗の旧社屋（EP10で発見。ARC2で拠点化）

# キャラクターの変化

- **奏**: 一人の調査者→三人のリーダーへ。ただし「リーダーになる」意図はまだない
- **桐嶋**: 奏を利用しつつ監視→不完全な信頼の始まり
- **詩音**: 引きこもりの孤独→「処理しやすい人間」として奏を受け入れる

# メモ

- EP001は静かに始める。封筒を読む奏の表情が変わらないことで、彼女の性格を一発で見せる
- EP008（奏が桐嶋の調査を知っていた）が二人の関係の転機。台詞は最小限に
- EP019-020（侵入→失敗→再挑戦→成功）で第1巻のクライマックス。「勝利ではなく存在の証明」
