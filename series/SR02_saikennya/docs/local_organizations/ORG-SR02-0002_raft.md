# 概要

RAFTは、調停機構に対抗していた組織。3年前に機構の内部工作で瓦解。メンバーは全員社会的に抹殺されるか行方不明になった。

# 瓦解の真相

**内部の裏切りではなかった。**

- リーダーが機構と交渉して**共存する道**を選ぼうとしていた
- それを知らない一人のメンバーが「裏切り」と誤解して内部告発した
- 結果、機構に組織の全貌を把握された
- **善意と誤解が組織を壊した**

この真相はARC2（EP30-32）で段階的に明らかになり、奏の組織運営の教訓として機能する。

# 物語における役割

- 奏たちの「先輩組織」としての反面教師
- 「善意が組織を壊す」というテーマの象徴
- 詩音が引きこもり中の2年間に瓦解の情報を独自収集していた（詩音の隠れた貢献）
- RAFTの失敗を踏まえて、奏は「調停機構と同じ手は使わない」というルールを明文化する（EP33）
