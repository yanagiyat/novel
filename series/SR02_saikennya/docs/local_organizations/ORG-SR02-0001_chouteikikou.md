# 概要

調停機構（通称：機構）は、戦後の混乱期にGHQと旧内務省の残党が共同設立した調整機関。冷戦期には左翼運動封じ込めに機能した。設立当初のイデオロギーはすでに失われ、現状維持そのものが目的化している。暗殺や拉致ではなく、信用の破壊・経済的締め上げ・人間関係の崩壊工作で人を消す。暴力を使わないため証拠が残らず、被害者すら人生が壊れた理由に気づかない。

# 表の顔

- 表向きの法人格は持たない
- 複数のコンサルティング会社、非営利団体、財団を迂回ルートとして使用
- 資金は複数の迂回ルートで隠蔽され、すべて合法的に処理されている

# 裏の顔

- ターゲット一人にチームが組まれる分業制（「案件」と呼ばれる）
- 担当者は全体像を知らされない
- 手口：偽証拠の捏造、SNS工作、広告引き揚げ圧力、金融機関への働きかけ、人間関係の切断工作
- 被害者は「運が悪かった」「自業自得」と思い込まされる

# 歴史

- **戦後**: GHQと旧内務省の残党が共同設立
- **冷戦期**: 左翼運動封じ込めに機能
- **高度経済成長期〜現代**: イデオロギーは失われ、現状維持が自己目的化
- **3年前**: 対抗組織RAFTを内部工作で瓦解させた

# 組織構造

| 層 | 名称 | 役割 |
|----|------|------|
| 上層 | 評議部 | 大手財団・業界団体の役員。互いの顔すら知らない |
| 中層 | 執行部 | 情報集約と方針決定 |
| 下層 | 現場部 | 実際の工作担当。来栖の元所属 |

# 関係勢力

- P25ML共有世界の特管局（ORG-G-0006）とは、存在を互いに認知している可能性がある（本編では触れない）
- 調停機構は超常現象には関与しない。純粋に世俗的な秘匿組織
- RAFTを瓦解させた実績がある

# 不都合な真実

**物語終盤で浮かぶ事実**: 機構がなければ現在の日本社会の安定はなかったという側面がある。単純な善悪にならない設計。

# 備考

- 単純な悪として描かないこと。組織の論理と存在理由を常に持たせる
- 「暴力を使わない」ことがこの組織の恐ろしさの核心。証拠が残らない
- 内部に別の割れ目がある（ARC4で判明）。外部の敵ではなく内部の矛盾も描く
