# 概要

御手洗が編集長を務めていた地方紙の旧社屋。廃刊後も取り壊されずに残っている。千葉県内の地方都市に所在。段ボールに入った30年分の取材資料と古い印刷機がある。ARC2（EP26）以降、奏たちの組織の拠点として整備される。

# 表の構造

- 2階建ての古い事務所ビル
- 1階：元編集室。作戦会議やデスクワークに使用
- 2階：元記者室。詩音の作業スペースと仮眠室
- 地方都市にあるため、東京からのアクセスは時間がかかるが、目立たない

# 裏の構造

- 特になし。普通の旧社屋
- ただし御手洗の30年分の取材資料が眠っており、情報的な価値は高い

# 注意事項

- 「組織のリアルな摩擦」の舞台として機能させる
- 豪華な秘密基地ではなく、段ボールと古い机の上で作戦が立てられる地に足のついた拠点
- 五人の日常（食事、雑談、沈黙）を描く場所としても重要
