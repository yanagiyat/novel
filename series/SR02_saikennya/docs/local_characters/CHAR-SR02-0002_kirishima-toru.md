# 概要

44歳、男性。元公安警察。調停機構の存在に気づいた時点でハメられ、汚職刑事として失職。現在は新宿の雑居ビルで違法すれすれの身辺調査屋を営む。信条は「信じるな、記録しろ」。

# 外見（EP004で確定）

- 四十代半ば、短髪、痩身
- ジャケットにジーンズ、スニーカー
- 顔は彫りが深く、目の下に疲労の影
- EP003での隣人の証言（40代、ジャケットにジーンズ、丁寧な話し方）と一致

# 口調

- 一人称：「俺」
- 短い文。余計なことを言わない
- 質問には質問で返すことが多い
- 感情を声に出さないが、行動に出る

# 能力・技能

- 公安警察時代の尾行・監視・情報収集技術
- 身辺調査のプロフェッショナル
- 対人観察力が高い（奏の欠点を補う位置）
- 物理的な自衛能力もある程度持つ

# 対人関係

- **奏**: 最初は協力しながら並行して身辺調査をする。奏がそれを知った上であえて何も言わないシーンが二人の関係の転機
- **来栖**: 最後まで完全には信用しない。しかし来栖の行動を見て態度が変化する
- **娘**: 元妻のもとに一人いる。巻き込まないために縁を切ってある。本音と建前の境界が本人にもわからない
- **元妻**: 縁を切るよう徹底してある

# 知っていること

- 公安時代に掴んだ調停機構の断片的情報
- 自分がハメられた手口の詳細
- 裏社会の情報ネットワーク

# 禁止事項

- 「信じるな、記録しろ」の信条を一貫させること
- 感情を台詞で直接語らせない。行動で示す
- 娘のエピソード（ARC4で機構に接触される）は最大の感情的シーンとして温存する

# キャラクターアーク

1. **初期**: 奏を利用しつつ監視する両面性
2. **転機**: 奏が監視を知っていたと告げても何も言わない——信頼の始まり
3. **中期**: 組織の中で参謀的役割を担う
4. **後期**: 娘が機構に接触され、初めて感情を崩す。チームが一人のために動く経験
5. **最終**: 娘と短い電話。台詞は短い

# 伏線（本人登場前）

- **SR02-EP003「転居先不在」**: 奏が宮田の旧住所（埼玉県北部）の隣家を訪問した際、「先月、同じ質問をした男」がいたと判明。特徴：40代、ジャケットにジーンズ、丁寧な話し方、身辺調査関連の名刺を渡した。宮田の昔の知り合いを自称。——桐嶋徹の設定（44歳、身辺調査屋）と一致。奏はEP003時点で正体に気づいていないが、「元公安」の可能性を推理している。

# 登場履歴

- **SR02-EP003**: 直接登場なし（上記「伏線」参照）
- **SR02-EP004**: 初登場。新宿歌舞伎町の雑居ビル4階「桐嶋調査事務所」にて奏と対面。以下が判明:
  - 7年前に公安で機構の存在に気づきハメられた（汚職刑事として処分、退職金なし）
  - 元妻と子どもは実家に帰し、連絡は取っていない
  - 機構に壊された人間のリストを独自に作成中
  - 奏の3ヶ月前に同じ匿名の封筒を受け取っている
  - 「信じるな、記録しろ」の信条を直接発言
  - 奏との関係: 組まないが情報交換はする。メモは手書き・万年筆
- **SR02-EP005**: メールで独自リスト（PDF、パスワード付き）を奏に開示。以下が判明:
  - **桐嶋のリスト**: 機構に壊された人間を独自に7人分リスト化した手打ちの表。1人目＝奏、2〜5人目＝封筒の資料と一致、6人目＝厚労省元係長（処理「進行中？」）、7人目＝欄のみ存在（名前・所属なし）
  - 奏の現在の状態を「湊裏区・翻訳下請け・単独行動中」と正確に把握
  - 六人目への接触を「善意の介入が最悪の結果を招く」として制止
  - 封筒の差出人について奏と同じ仮説（受取人の反応を観察してから次を選んでいる）に到達
  - 共通点の指摘：「調査能力。そして機構に壊されても、立ち直った人間」
