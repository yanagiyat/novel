# 概要

44歳、男性。元公安警察。調停機構の存在に気づいた時点でハメられ、汚職刑事として失職。現在は新宿の雑居ビルで違法すれすれの身辺調査屋を営む。信条は「信じるな、記録しろ」。

# 外見

（執筆時に確定。中年の身辺調査屋らしい地味さと、元公安の鋭さの同居を推奨）

# 口調

- 一人称：「俺」
- 短い文。余計なことを言わない
- 質問には質問で返すことが多い
- 感情を声に出さないが、行動に出る

# 能力・技能

- 公安警察時代の尾行・監視・情報収集技術
- 身辺調査のプロフェッショナル
- 対人観察力が高い（奏の欠点を補う位置）
- 物理的な自衛能力もある程度持つ

# 対人関係

- **奏**: 最初は協力しながら並行して身辺調査をする。奏がそれを知った上であえて何も言わないシーンが二人の関係の転機
- **来栖**: 最後まで完全には信用しない。しかし来栖の行動を見て態度が変化する
- **娘**: 元妻のもとに一人いる。巻き込まないために縁を切ってある。本音と建前の境界が本人にもわからない
- **元妻**: 縁を切るよう徹底してある

# 知っていること

- 公安時代に掴んだ調停機構の断片的情報
- 自分がハメられた手口の詳細
- 裏社会の情報ネットワーク

# 禁止事項

- 「信じるな、記録しろ」の信条を一貫させること
- 感情を台詞で直接語らせない。行動で示す
- 娘のエピソード（ARC4で機構に接触される）は最大の感情的シーンとして温存する

# キャラクターアーク

1. **初期**: 奏を利用しつつ監視する両面性
2. **転機**: 奏が監視を知っていたと告げても何も言わない——信頼の始まり
3. **中期**: 組織の中で参謀的役割を担う
4. **後期**: 娘が機構に接触され、初めて感情を崩す。チームが一人のために動く経験
5. **最終**: 娘と短い電話。台詞は短い
