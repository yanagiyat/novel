# 概要

51歳、女性。元地方紙編集長。30年近く勤務し、後半10年を編集長として務めた。調停機構に不利な記事を出し続けた結果、広告を全部引き揚げられ廃刊に追い込まれた。怒りは完全に燃え尽きている。諦念と好奇心が同居している。

# 外見

（執筆時に確定。50代の落ち着いた外見。段ボールに囲まれた旧社屋で過ごす姿）

# 口調

- 一人称：「私」
- 落ち着いた、しかし少し皮肉を含んだ話し方
- 長い沈黙を厭わない
- 奏への最初の言葉：「あなたには無理です」——直後に「それでも聞きますが」

# 能力・技能

- 30年の取材経験と蓄積された情報・人脈
- ジャーナリストとしての調査・検証スキル
- 業界の旧コネクション（現役の記者・編集者との関係）
- 文章力。記事として世に出す力
- **傷**: 過去に一度スクープを握りながら家族への影響を考えて記事にしなかった

# 対人関係

- **奏**: 年長者として見守る。参加理由を語らない
- **詩音**: 最初は世代差で衝突。共同作業を通じて和解。ARC3で詩音が崩れた時に話しかける
- **桐嶋**: 大人同士の静かな了解がある
- **来栖**: 合流に唯一賛成する。その理由は明示しない
- **家族**: スクープを出さなかった理由が家族。現在の関係は不明

# 知っていること

- 30年分の取材で蓄積した調停機構に関する情報
- 出さなかったスクープの内容
- メディア業界の構造と、広告主による支配の手口

# 禁止事項

- 「優しいおばさん」にしないこと。諦念と好奇心と皮肉の同居を維持
- 号泣シーンは一度だけ。三十年分の重さとして機能させる。安易に泣かせない
- 出さなかった記事の内容はARC5（EP88）まで温存する

# キャラクターアーク

1. **初期**: 廃刊した旧社屋で段ボールを整理する老いた背中。燃え尽きている
2. **転機**: 奏の提案を断り、しかし「それでも聞きますが」と続ける
3. **参加**: 理由を語らないまま加わる。拠点として旧社屋を提供
4. **中期**: 詩音との世代を超えた関係。桐嶋との大人の了解
5. **クライマックス**: 三十年前の出さなかった記事をついに出す（EP88）
6. **号泣**: 一度だけ、公衆の面前で。三十年分
