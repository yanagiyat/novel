# 概要

農林水産省の元係長。調停機構による社会的抹殺の三人目の被害者。奏が封筒の資料から追跡している人物の一人。

# 判明している情報

- **名前の出典**: 資料では黒塗りだったが、官報の在籍記録から奏が「宮田」と推定（SR02-EP003）
- **旧住所**: 埼玉県北部の住宅地（最寄り駅から徒歩20分）
- **家族構成**: 妻、子供二人（上の子が中学入学前＝約12歳だった）
- **退職後の経緯**: 仕事を辞めた後、挨拶もなく突然転居。近所の誰も転居先を知らない
- **転居時期**: 約6年前（上の子の中学入学前）
- **家屋の現状**: 今年（物語時点）4月に解体され更地。売地看板あり。敷地は約50坪
- **最後の目撃**: 引っ越し1週間前の夜遅く、暗い庭で携帯電話で約30分間通話していた（隣家の女性の証言）

# 人物像

- 無口な人物（隣家の女性がほとんど会話したことがない）
- 妻は近所付き合いがあった（井戸端で隣家の女性と話していた）

# 物語上の機能

- 奏が調停機構の被害パターンを分析する材料の一つ
- この住所への聞き込みで「先客」（桐嶋徹の伏線）の存在が判明する起点
