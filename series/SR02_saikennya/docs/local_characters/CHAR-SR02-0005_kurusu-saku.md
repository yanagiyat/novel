# 概要

35歳、男性。元調停機構現場部クリーナー。ターゲットの信用を破壊するための偽証拠を作り、SNSに流し、周囲との関係を切らせる——暴力は一切使わないが人生の消去に近い仕事をしていた。担当したターゲットの一人が自殺し、その娘が泣いている動画がSNSに流れたことで離脱を決意。しかし完全に離脱できておらず、機構とチームのどちら側にもいられない半端な状態にある。

# 外見

（執筆時に確定。嘘をつく技術が高い人間の、常に整った外見を推奨）

# 口調

- 一人称：「僕」
- 穏やかで丁寧。しかしどこか距離がある
- 嘘と本当の境界が本人にもわからなくなっていることが、話し方に滲む
- 奏に対してだけ、時々不用意に正直になる

# 能力・技能

- **機構の手口と内部構造の知識**: 唯一の内部情報源
- **偽証拠の作成技術**: SNS工作、信用破壊の手法に精通
- **対人操作**: 人の信用を得る技術が極めて高い。裏を返せば、何が本音かわからない
- **機構の裏金**: 離脱時に持ち出した資金がある

# 対人関係

- **奏**: 正直に話すシーンが来栖の核になる。奏だけに本音を見せる（本当かどうかは不明）
- **桐嶋**: 最後まで完全には信用されない。しかし来栖の行動を見て態度は変化する
- **御手洗**: 合流に唯一賛成してくれた人。その意味を来栖は噛みしめている
- **機構の旧知**: EP71で「戻ってこい」という接触を受ける。それを奏に報告する

# 知っていること

- 調停機構の現場部の手口の全容
- 機構の中間層（執行部）の一部の人物情報
- 資金ルートの一端
- 自分が壊した人々のリスト

# 禁止事項

- ARC3終盤まで信用できるかどうかを読者に断定させないこと
- 単純な裏切り者にも、単純な味方にもしないこと。灰色を維持
- 嘘をつく技術を便利に使いすぎない。本人が自分の本音がわからないという苦しみを描く
- EP71（旧知からの接触→奏に報告）は「隠さなかったことの意味」を問う重要シーン

# キャラクターアーク

1. **登場前**: 情報として存在が浮かぶ（EP34）。元クリーナーという事実が緊張を生む
2. **接触**: 奏が単独で会いに行く。来栖は逃げない（EP36）
3. **合流**: 桐嶋は保留、詩音は沈黙、御手洗だけが賛成（EP38）
4. **裏金問題**: 持ち出した機構の裏金を奏が「借りている金」として帳簿に記録（EP39-40）
5. **信用危機**: ARC3で機構に待ち伏せされ、来栖が原因か不明（EP52-54）
6. **証明**: 単独行動でリーク元が別ルートだったと証明。信用と傷が同時に残る（EP59-60）
7. **試練**: 機構の旧知から戻ってこいという接触。隠さずに報告（EP71-72）
8. **終盤**: 過去の案件の被害者と対面。自分の罪を外に向かって語る（EP85-86）
9. **結末**: 決着ではなく継続としての終わり方（EP98）
