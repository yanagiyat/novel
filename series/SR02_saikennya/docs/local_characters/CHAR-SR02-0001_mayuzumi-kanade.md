# 概要

29歳、女性。元国税局調査官。上司の横領を内部告発しようとして逆に証拠を捏造され懲戒免職。その後、調停機構の工作により職も家族関係も失った。失ったものへの怒りよりも、自分の人生がどのような仕組みで壊されたかを解明したいという知的執念で動く。

# 外見

（執筆時に確定。地味で目立たない外見を推奨。職業人としての清潔感は残る）

# 口調

- 一人称：「私」
- 簡潔。必要なことだけを話す
- 感情が高ぶっても声のトーンが変わりにくい
- 質問の形で相手を動かすことが多い

# 能力・技能

- **戦闘力ゼロ**。物理的な力は一切ない
- **帳簿から嘘を読む目**: 元国税局調査官としての専門能力。数字の矛盾から全体像を組み立てる
- **壊される前の人間関係の広さ**: 元同僚、元取引先など。ただし機構の工作で多くは断たれている
- **構造的思考**: 怒りを即座に「では何をすれば状況が変わるか」に変換する癖
- **欠点**: 人の感情的なサインを見逃しがち。組織が揺れる局面でこの欠点が出る

# 対人関係

- **桐嶋（CHAR-SR02-0002）**: 互いに手の内を見せない関係から始まり、不完全な信頼を築く。最も近い参謀
- **永瀬 詩音（CHAR-SR02-0003）**: 対人能力を問題にしないため、詩音にとって「処理しやすい人間」。気づいたら信頼されていた
- **御手洗（CHAR-SR02-0004）**: 30年分の重みを持つ先輩。御手洗の決断を尊重する
- **来栖（CHAR-SR02-0005）**: 最も危険で最も必要な存在。信用問題を抱えながら組織に組み込む
- **父親**: 税務署員。正しいことをして左遷された。奏が「正義より構造に興味を持つ人間」に育った原点

# 知っていること

- 自分の人生が壊された手口の詳細（部分的）
- 国税局時代の財務分析スキルで読み解ける調停機構の資金ルートの断片
- RAFTの存在（調査を通じて発見）

# 知られていること

- 調停機構からは「処理済み案件」として認識されている（最初は）
- 元同僚からは「問題を起こして辞めた人」として避けられている

# 禁止事項

- 万能感を持たせないこと。感情的サインを見逃す欠点を定期的に出す
- 感情の爆発を安易に使わないこと。崩れるタイミングは執筆中に最も自然な場所に置く
- 「調停機構と同じ手は使わない」というルールを物語の核心として機能させること

# キャラクターアーク

1. **初期（ARC1）**: 一人で真相を追う孤独な調査者
2. **中期（ARC2-3）**: 組織のリーダーとしての成長。ルールの明文化
3. **転換（ARC4）**: 守るもの（仲間）が生まれたことで、ルールの維持が困難になる
4. **最終（ARC5）**: 同じ手を使いかけるが、チームに止められる。「組織としての強さ」に着地
