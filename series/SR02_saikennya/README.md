# SR02: 再建屋

## ジャンル
クライム・サスペンス / 群像劇 / 社会派

## コンセプト
戦後から暗躍する秘匿組織「調停機構」に人生を壊された者たちが、証拠の残らない敵にどう戦うかを描く。
暴力ではなく信用破壊で人を消す敵に対し、「同じ手は使わない」という原則を貫けるかが核心。

## 文体
パターン2（ハード＆ロジカル文体）ベース。ただし基本は奏の一人称視点。
群像スタイルの章で視点キャラを変え、奏が他者からどう見えているかを多角的に描く。
感情は行動と観察を通じて間接的に描写。抑制が効いた静かな緊張感を基調とする。

## 主要キャラクター

- **黛 奏**（29歳・女）: 元国税局調査官。帳簿から嘘を読む。戦闘力ゼロ。組織のリーダー
- **桐嶋 徹**（44歳・男）: 元公安警察。「信じるな、記録しろ」
- **永瀬 詩音**（23歳・女）: 元ITエンジニア。OSINT技術者。対面が苦手
- **御手洗 幸**（51歳・女）: 元地方紙編集長。30年分の未出の記事を持つ
- **来栖 朔**（35歳・男）: 元調停機構クリーナー。唯一の内部情報源

## P25ML共有世界との接点

- world_exposure: low — 裏設定は背景ノイズ程度
- 調停機構は「秘匿結社」系統に位置づけ。ただし超常要素には関与しない世俗組織
- 共有インフラ企業（エンリング、アルカID等）は日常背景として使用可能
- 特管局との関係は「互いに存在を認知しているが干渉しない」程度（本編では触れない）

## 執筆時の注意

- 調停機構を単純な悪として描かない。物語終盤で「機構がなければ現在の日本社会の安定はなかった」という不都合な事実を浮かべる
- 来栖が信用できるかどうかを読者に断定させない（少なくともARC3終盤まで）
- 奏に万能感を持たせない。感情的サインを見逃す欠点を定期的に出す
- 作戦を毎回成功させない。失敗と撤退も物語の駆動力にする
- 感情の爆発を多用しない。御手洗の一度の号泣のみ例外
