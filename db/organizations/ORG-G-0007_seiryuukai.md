# 概要

星流会（Seiryuukai）は、表向きには複数業界の経営者が集う異業種交流会・経済団体だが、実態は異常事象に関わる民間企業や有力個人を緩やかに束ねる連合体。国家機関（特管局等）とは独立した立場で、民間側の秩序維持と利益保護を図る。

# 表の顔

- 年に数回の経営者フォーラムやセミナーを開催する一般的な経済団体
- 会員企業は中堅〜大手の優良企業が多い
- 慈善活動や奨学金事業も行っている
- 特に目立つ活動はなく、注目されることも少ない

# 裏の顔

- 異常事象に何らかの形で関わる企業のオーナー・経営幹部が中核を構成
- 玄庫ロジスティクス、翼医療グループ、アルカID等のインフラ企業も会員に含まれる
- 国家機関の統制に対する民間側のカウンターバランスとして機能
- 異常事象に関する情報共有、相互扶助、利益調整を行う
- 統一された指揮系統はなく、合議と利害調整で動く

# 歴史

- 起源は明治期の実業家たちの私的な集まりに遡る
- 戦後、GHQ占領下で一度解散状態になるが、1960年代に再結成
- 高度経済成長期に勢力を拡大
- 現在は第三世代の経営者たちが中心

# 組織構造

- **幹事会**: 主要メンバー12名による意思決定機関
- **事務局**: 情報収集・連絡調整
- 各会員企業は独立して活動し、星流会としての統一行動は稀

# 関係勢力

- 特管局とは協力と対立の両面がある
- 会員企業を通じて、多くの秘匿勢力と間接的な接点を持つ
- 犯罪組織とは一線を画しているが、灰色の取引を完全には排除していない

# 備考

- 善悪の単純なレッテルは貼れない。民間の利益と秩序維持のバランスを取ろうとしている
- 日常シリーズでは「あの会合」「業界の集まり」程度に言及されることがある
