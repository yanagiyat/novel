# 概要

翼医療グループ（Tsubasa Medical Group / TMG）は、首都圏を中心に12の病院と23のクリニックを運営する民間医療法人グループ。1987年設立。総合病院からバイオ研究施設まで幅広い医療インフラを持つ。旗艦病院「翼中央病院」は霞崎区に所在。

# 表の顔

- 首都圏最大級の民間病院ネットワーク
- 先端医療研究（再生医療、遺伝子治療、AI診断）で知られる
- 救急医療の受け入れ体制が充実
- 医療系スタートアップへの投資・インキュベーション事業

# 裏の顔

- 「通常の医療では説明できない傷病」を専門に扱う非公開部門が存在
- 異常事象の影響を受けた人体の治療と研究を行っている
- 患者の中には、公式記録に残らない形で入院している者がいる
- バイオ研究部門の一部は、人体と異常現象の相互作用を研究対象としている

# 歴史

- 1987年: 創業者・翼 宗一郎が霞崎区に翼病院を開設
- 1995年: 阪神大震災の医療支援を契機に災害医療に注力
- 2005年: グループ化。複数の病院・クリニックを傘下に
- 2015年: バイオ研究施設「翼ライフサイエンスセンター」を湊裏区に開設
- 2023年: 非公開部門の設立（公式記録なし）

# 組織構造

- **表の部門**: 各病院の診療科、研究開発部、経営管理部
- **裏の部門**: 特殊症例研究室（場所非公開、関係者にも所在が完全には明かされない）

# 関係勢力

- 表向きは独立した民間医療法人
- 一部の公的機関から「特殊な患者」の紹介を受けている
- 秘匿結社や非公式組織からの協力要請を受けることもあるが、選択的に対応

# 備考

- 日常シリーズでは普通の病院として登場可能
- 医療系・怪異系のシリーズでは重要な舞台になりうる
