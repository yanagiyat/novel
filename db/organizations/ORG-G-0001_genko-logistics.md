# 概要

玄庫ロジスティクス（Genko Logistics / GKL）は、全国規模の高機能配送ネットワークを運営する物流企業。2018年設立。即日配送と特殊貨物対応を強みに急成長し、大手宅配業者に次ぐ規模を持つ。本社は湊裏区ロジスティクスパーク内。社名の「玄」は奥深さ、「庫」は蔵を意味する。

# 表の顔

- 全国に300以上の拠点を持つ配送ネットワーク
- 個人向け即日配送から法人向け特殊貨物まで幅広く対応
- AIによる配送最適化システムが業界で注目されている
- 若い創業者と先進的な企業文化で採用市場でも人気

# 裏の顔

- 「特殊貨物対応」の一部は、通常の物流では扱えない異常物件の封緘・移送を含む
- 各秘匿勢力の荷物を中立的立場で受け付ける「灰色の運び屋」としての機能を持つ
- 配送ドライバーの一部は、異常物件に対する耐性審査を通過した特別要員
- 玄庫ロジスティクス自体は特定の勢力に属さず、中立性を最大の資産としている

# 歴史

- 2018年: 創業。当初は都内限定の即日配送サービス
- 2020年: コロナ禍で需要急増。全国展開開始
- 2022年: 特殊貨物部門を正式に設立（表向き）
- 2024年: 湊裏区に本社移転。ロジスティクスパーク内に巨大拠点を構築

# 組織構造

- **表の部門**: 一般配送部、法人営業部、システム開発部、管理部
- **裏の部門**: 特殊案件課（社内でも限られた人間しか存在を知らない）

# 関係勢力

- 中立を維持しているため、多くの勢力と取引関係がある
- 特定勢力への肩入れは組織の存続を脅かすため、厳格に禁止されている
