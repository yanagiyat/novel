# 概要

アルカID（ARCA-ID / Arca Identity）は、個人認証と電子決済を統合した民間プラットフォームサービスを運営する企業。2020年設立。マイナンバーカードと連携した本人確認、電子決済、行政手続きの一元化を実現し、都市生活のインフラとして急速に普及。利用者数3000万人超。

# 表の顔

- マイナンバーカード連携の本人確認サービス
- QRコード・NFC対応の統合電子決済「ArcaPay」
- 行政手続きのオンライン化支援
- 法人向け本人確認API（KYC）の提供
- ポイント・特典プログラム

# 裏の顔

- 膨大な取引データと行動パターンから、「通常から逸脱した個人の行動」を検知するAIを運用
- 検知対象には、異常物件の購入パターン、不審な移動パターン、特定の施設への頻繁なアクセスが含まれる
- 検知結果は特定の公的機関にのみ共有され、一般のマーケティングには使用されない
- プライバシー問題のリスクを認識しつつも、「異常の民間流出」を防ぐ名目で運用が継続

# 歴史

- 2020年: 設立。マイナンバーカード普及政策の追い風で成長
- 2022年: ArcaPay開始。コンビニ・交通機関との連携で利用者急増
- 2024年: 利用者3000万人突破。事実上の社会インフラ化
- 2025年: 行動パターン検知システムの運用開始（非公開）

# 組織構造

- **表の部門**: プロダクト開発、決済インフラ、法人営業、コンプライアンス
- **裏の部門**: データ解析室内の特別プロジェクト（社内でも数人しか知らない）

# 関係勢力

- 政府のデジタル行政推進と連携
- 一部の公的機関と非公開の情報共有関係がある
- 金融業界・小売業界との幅広い提携
