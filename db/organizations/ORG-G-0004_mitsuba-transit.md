# 概要

三葉トランジット（Mitsuba Transit / MTR）は、東京都心部の地下鉄新規路線および地下都市インフラを管理する半官半民の交通事業体。2019年設立。既存の地下鉄事業者とは別に、新たな都心環状地下路線と深層地下インフラの建設・運営を担う。

# 表の顔

- 都心環状地下路線「Mライン」の建設・運営（2026年部分開業予定）
- 深層地下通路・地下広場の整備
- 災害時の地下避難施設の管理
- 地下空間の有効利用に関する研究開発

# 裏の顔

- 地下掘削の過程で発見された「説明できない空間」の管理を担っている
- 一般路線として運用されない地下区間が複数存在する
- 地下ネットワークを通じた秘匿移動ルートを一部の認可を受けた組織に提供
- 地下深層で観測される異常な計測値の監視・記録

# 歴史

- 2019年: 東京都と民間企業群の出資で設立
- 2021年: 都心環状地下路線「Mライン」の掘削開始
- 2022年: 掘削中に「予定外の空洞」を複数発見。工事が一時中断
- 2023年: 発見された空洞の管理を含む業務拡張。組織体制が変更される
- 2025年: Mライン一部区間の試験運行開始

# 組織構造

- **表の部門**: 路線建設部、運行管理部、施設管理部、安全対策部
- **裏の部門**: 深層管理課（地下異常空間の監視・管理）

# 関係勢力

- 東京都が筆頭出資者だが、民間企業も多数参画
- 深層管理課の活動について、一部の国家機関と情報共有している
- 地下空間の利用を巡り、複数の勢力から関心を寄せられている

# 備考

- 日常シリーズでは「新しい地下鉄が工事中」という背景として登場可能
- 地下探索系・都市伝説系のシリーズでは中心的な舞台になりうる
