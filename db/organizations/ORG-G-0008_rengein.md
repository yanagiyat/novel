# 概要

蓮華院ネットワーク（Rengein Network）は、表向きには全国の仏教系寺院が緩やかに連携する宗教法人ネットワーク。実態は、古来からの霊的技術（陰陽道、密教、修験道等を含む）を継承する宗教者・術者の連合体。怪異や異界の浸食から社会を守ることを使命としている。

# 表の顔

- 全国約120の寺院が加盟する仏教系連携ネットワーク
- 文化財保護、地域貢献、葬祭サービスが主な活動
- 仏教系大学との学術交流も行っている
- 穏やかな宗教組織として地域に溶け込んでいる

# 裏の顔

- 加盟寺院の中でも「実働」に関わるのは30寺院程度
- 各地の霊的な要所（霊脈の結節点、封印地、異界の薄い場所）を代々管理
- 怪異の鎮静化、異界浸食の封じ込め、霊的被害者の保護を行う
- 陰陽道・密教・修験道など複数の霊的技術体系を併用
- 後継者不足に悩んでおり、技術の断絶が深刻な問題になっている

# 歴史

- 起源は平安期にまで遡る（記録は断片的）
- 江戸期に現在に近い組織形態が確立
- 明治の廃仏毀釈で大きな打撃を受けるが、地下に潜って存続
- 戦後に宗教法人として再組織化
- 現代では世俗化と後継者不足が最大の課題

# 組織構造

- **大僧正会**: 最高意思決定機関（5名）
- **実働衆**: 現場で活動する術者・僧侶
- **管理衆**: 各地の管理拠点の維持
- **養成衆**: 後継者の育成

# 関係勢力

- 特管局とは古くからの関係があるが、信頼関係は複雑
- 星流会の一部メンバーとも交流がある
- 神道系の組織とは協力関係にあるものもあれば、教義上の対立があるものもある
- 古代文明の遺構に関しては独自の解釈を持っている

# 備考

- 善悪の判断は単純ではない。「守る」ことが常に正しい行動とは限らない場面もある
- 日常シリーズでは「お寺」として自然に背景に溶け込む
- 怪異系・陰陽師系のシリーズでは主要勢力になる
