# 概要

湊裏区は東京湾岸に位置する架空の特別区。江東区南部から品川区東部にかけての埋立地エリアに相当する。巨大物流倉庫、コンテナターミナル、工業施設が集積する一方、2010年代以降のタワーマンション群も立ち並ぶ。人口約15万人だが、昼間人口は物流・港湾従事者を含め大きく膨らむ。

# 表の構造

- **湊裏コンテナターミナル**: 東京湾岸有数のコンテナ取扱量を持つ埠頭
- **湊裏ロジスティクスパーク**: 複数の物流企業の巨大倉庫が並ぶ一帯
- **シーフロントタワー群**: 湾岸の新興高層マンション地帯。若年ファミリー層が多い
- **湊裏水産市場**: 中規模の水産市場。周辺に飲食店が集まる
- **湊裏工業団地**: 中小の製造業・加工業が集まる

# 裏の構造

- コンテナターミナルの一部は、通常の通関手続きを経ない特別ルートが存在する
- ロジスティクスパークの地下に、一般の搬入出記録に載らない保管区画がある
- 埋立地の最下層に、明治期の埋立時に封じ込められたとされる構造物の痕跡がある
- 湾岸の一部水域に、計器が異常を示す海域がある

# 立入条件

- 住宅地・商業地は一般人が自由に出入り可能
- コンテナターミナル・工業団地は関係者以外立入禁止の区画がある
- 地下構造へのアクセスは一般に知られていない

# 注意事項

- 物流・港湾の雰囲気を重視すること
- 湾岸タワマンの生活感も描けるので、日常シリーズの舞台としても使用可能
- 海と倉庫と夜景というビジュアル要素を活かせる
