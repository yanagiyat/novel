# 共通ステータス値

すべてのエンティティの `status` フィールドで使用する許容値。

## ステータス定義

| ステータス | 説明 | 用途 |
|-----------|------|------|
| active | 現在活動中・存在中 | 人物・組織・アイテム・場所すべて |
| inactive | 活動停止中だが存在はしている | 休眠中の組織、引退した人物など |
| missing | 行方不明・所在不明 | 人物・アイテム |
| sealed | 封印されている | アイテム・場所・存在 |
| retired | 引退・退役 | 人物・組織の役職 |
| deceased | 死亡・消滅 | 人物・存在 |
| unknown | 状態不明 | あらゆるエンティティ |
| suspended | 一時停止・凍結 | 組織の活動、プロジェクト |
| deprecated | 廃止・無効化（削除の代替） | あらゆるエンティティ |

## 運用ルール

- 削除は原則禁止。不要になったエンティティは `deprecated` に変更する
- `deprecated` にする際は理由を `notes` に記録すること
- `deceased` は物語内での死亡を意味し、メタ的な削除には使用しない
- `unknown` は設定が未確定の場合にも使用可能
