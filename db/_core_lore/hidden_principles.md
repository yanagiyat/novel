# 確定事項

## 秘匿の多層性
- 世界の裏側は単一の「真実」として管理されているのではなく、複数の層がある
- ある組織にとっての「裏の真実」が、別の組織にとってはまだ表層に過ぎない場合がある
- 完全な真相を知る存在がいるかどうかは未確定

## 秘匿の維持主体
- 秘匿を維持する主体は複数存在し、それぞれの動機と方法が異なる
  - 国家機関系: 社会秩序の維持、パニック防止
  - 民間系: 利権の保護、技術独占
  - 非人間系: 自らの存在の隠蔽、干渉の最小化
  - 古代系: 太古の盟約や禁忌に基づく秘匿
- 各主体間で協力もあれば対立もある

## 露出のグラデーション
- すべてが完全に隠されているわけではない
- 都市伝説、オカルト雑誌、ネットの噂として断片的に漏れている情報がある
- 「知っている人は知っている」レベルの情報から、真に封印された情報まで段階がある

# 仮説

- 秘匿が完全に崩壊したらどうなるか（ディスクロージャー問題）は保留
- 過去に秘匿が大規模に破れた事例があったかどうかは保留

# 保留

- 秘匿維持のための最上位組織・最上位存在の有無
- 「隠す必要がなくなる未来」の可否
