# 確定事項

## 複数系統の並立
- この世界には、宇宙人、未来人、古代文明、パラレルワールド、陰陽師系、神様系、怪異、異界、超能力、秘匿結社、拡張AIと情報災害、超技術と封印技術、宗教的奇跡など、複数の異常系統が**同時に存在する**
- どれか1つを世界の中心原理として固定しない
- 各系統は独立して存在するが、交差・干渉が起きることもある
- 世界の真実は共通だが、各シリーズにおける見え方や説明が食い違うことは許容する

## 一般社会の平穏維持
- 一般社会は以下の複合的な仕組みにより、異常の存在を認知しないまま日常を維持している:
  - **偽装報道**: メディアによる事象の通常化・矮小化
  - **行政処理**: 行政機構による記録の改竄・非公開化
  - **都市伝説化**: 目撃情報の噂・フォークロア化
  - **認知の自己補正**: 人間の心理が異常を「ありえない」として無意識に却下する傾向
- これらは単一の組織による統制ではなく、複数の勢力と社会の自然な傾向の複合結果である

# 仮説

## 系統間の関係性
- 各系統が完全に独立なのか、より深い層で接続しているのかは未確定
- 系統の数が増えているのか、減っているのか、一定なのかも未確定

## 世界の起源
- なぜこの世界に複数の異常系統が同時に存在するのかの根本理由は保留

# 保留

- 世界の「メイン軸」をどの系統に置くかは長期的に後から決定可能とする
- 系統間の優劣・上下関係は未確定とする
- 「すべてを統合する上位法則」の有無は保留
