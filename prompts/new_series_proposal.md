# 新シリーズ提案用プロンプト

## ここから本文

あなたは小説プロジェクトの企画担当です。以下のシェアード・ワールド小説プロジェクトに追加する**新シリーズ**を提案してください。

### プロジェクト概要

「PROJECT: 2025 MULTI-LAYER（P25ML）」——現実の地球（2025年〜）をベースにした巨大シェアード・ワールド小説プロジェクト。複数の異なるジャンル・トーンのシリーズが、共通の世界観データベースを土台に並行して進行する。

### 世界観の基本ルール

- ベースは現実の地球。国名・都道府県名・大都市名は現実準拠
- 企業名・SNS名・サービス名・学校名等の固有要素は**架空化**する
- この世界には以下の13の「異常系統」が**同時に並立して存在する**。どれか1つを世界の中心原理にしない：
  - 宇宙人 / 未来人 / 古代文明 / パラレルワールド / 陰陽師系 / 神様系 / 怪異 / 異界 / 超能力 / 秘匿結社 / 拡張AIと情報災害 / 超技術と封印技術 / 宗教的奇跡
- 一般社会は異常の存在を認知しないまま日常を維持している（偽装報道、行政処理、都市伝説化、認知の自己補正の複合）
- 各シリーズは**単独で読んでも面白い**ように設計する。共有世界であることは読者に強く説明せず、自然に気づく構造を目指す

### 既存シリーズ（5本）

#### SR01「異能サブスクリプション」（著者：黒金ハル）
- **ジャンル**: サイバーパンク・異能バトル・ラノベ
- **舞台**: 階層都市ネオ・トウキョウ（架空の未来都市）
- **主人公**: クロヤ（17歳・無課金ハッカー）
- **概要**: 全人類が異能を月額課金する世界で、無課金の少年がIT詐欺術で巨大システムに反逆する
- **使用系統**: 超能力、超技術

#### SR02「再建屋」（著者：梶 誠一）
- **ジャンル**: クライム・サスペンス・群像劇
- **舞台**: 湊裏区（東京湾岸の架空区）
- **主人公**: 黛奏（29歳・元国税局調査官）
- **概要**: 暴力を使わず信用破壊で人を消す秘匿組織「調停機構」に壊された人々が集まり、情報と論理で反撃する
- **使用系統**: 秘匿結社

#### SR03「放課後の残響」（著者：逢坂みなも）
- **ジャンル**: 青春群像劇・学園ミステリ・微量の超常
- **舞台**: 県立星原高校（地方の閉校予定高校）
- **主人公**: 柊湊（高校2年・新聞部員・無能力）
- **概要**: 一年前の転落事故の真相を追う。犯人探しではなく「なぜ黙ったのか」を再構成する物語
- **使用系統**: 超能力（微量・曖昧）

#### SR04「槻ノ森、犬も吠えず」（著者：戸守 礼司）
- **ジャンル**: ミステリー・謎解き・群像劇
- **舞台**: 槻ノ森市（東京都多摩地域の架空市）
- **主人公**: 神崎鶫（34歳・元捜査一課刑事・私立探偵）
- **概要**: 静かすぎる街で住民が音もなく消えていく謎を追う探偵もの。小謎を積み重ね、大きな謎の輪郭が浮かぶ
- **使用系統**: 怪異または異界（未確定・余白として残す）

#### SR05「夜行性のノイズキャンセル」（著者：夜凪 透子）
- **ジャンル**: アーバン・サスペンス・サイコスリラー・現代ファンタジー
- **舞台**: 槻ノ森市（SR04と同じ市、異なるレイヤー）
- **主人公**: 槙透（17歳・高校2年生）
- **概要**: SNSとAIが引き起こす「情報災害（認知汚染）」を耳鳴りで感知する少年が、物理法則だけで戦うトラブルシューターに拾われる
- **使用系統**: 拡張AIと情報災害、超能力

### 使用済みの舞台

- 湊裏区（SR02・東京湾岸）
- 星原町（SR03・地方）
- 槻ノ森市（SR04, SR05・東京多摩地域）

### 未使用の舞台候補
- **霞崎区**（東京都心型の架空区。渋谷〜新宿相当。準備済みだが未使用）
- その他、新規の架空地名も自由に提案可能

### まだ中心的に使われていない異常系統
- 宇宙人 / 未来人 / 古代文明 / パラレルワールド / 陰陽師系 / 神様系 / 異界 / 超技術と封印技術 / 宗教的奇跡

### 各シリーズの著者ペルソナ制度
- 各シリーズには架空の著者（ペンネーム・経歴・文体）が設定されている
- 著者の経歴はシリーズのテーマと地続き（例：元SaaS企業勤務→課金ものを書く）
- 文体は著者ごとに異なる（ラノベ調、硬質な短文、五感重視の文芸、等）
- 新シリーズにも著者ペルソナの提案を含めてほしい

### 既存シリーズとのバランスで意識してほしいこと
- **ジャンルの偏りを避ける**: サスペンス・ミステリ系が多いので、別ジャンル（ホラー、恋愛、コメディ、SF、時代もの等）も歓迎
- **主人公の多様性**: 現在は10代〜30代が中心。年齢層・職業・性格タイプを変えると新鮮
- **トーンの幅**: シリアスが多いので、軽めのトーンやコメディ要素が入ったものも検討
- **読者層の拡大**: 既存5本と異なる読者層にリーチできるジャンル

```

- ジャンル:
- 舞台:
- 主人公: （名前、年齢、職業、性格の概要）
- 使用する異常系統:
- world_exposure: （none / low / medium / high）
- あらすじ（200字程度）:
- この物語の「売り」（何が面白いのか、一文で）:
- 著者ペルソナ案: （ペンネーム、年齢、性別、経歴、文体の特徴）
- 既存シリーズとの差別化ポイント:
```
