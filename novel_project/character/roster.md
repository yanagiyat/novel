# 登場人物（※原則カタカナ日本名。別文化圏のみ海外名）

### メインキャラクター（3つの視点）
* **探偵：キジマ**（40代）
    * 治安維持局の枯れた捜査官。「目立ちすぎず、忘れられもしない」平均値を保つ。手巻き時計や紙の手帳など「アナログのノイズ」にのみ安堵を覚える。
* **復讐者：ジン**（30代）
    * 左腕が時折「ワイヤーフレーム化し強烈な幻肢痛を伴うバグ」を抱える潜入工作員。他人のIDを脳内ハッキングして「つまらない人間」を演じる存在論的迷彩の使い手。
* **恋人たち：タリクとエララ**（17歳・海外名）
    * 紛争地帯の少年少女。陣営の違う相手に触れると、システムが「敵」と判定し強烈な疑似痛覚（火傷）を与える。互いの素顔もノイズがかって見えない。

### サブキャラクター
* **レイ:** キジマの後輩捜査官。「いいね」の恩恵（疑似身体強化）を信奉する現代の若者。キジマと対立する。
* **シドウ:** 異端審問官。タリクたちを追う従軍インフルエンサー。信者の敵意を「物理的圧力」として行使するが、圏外ではただの男に成り下がる。
* **シズカ:** 声帯を自ら切り落としたIDブローカー。ジンに偽装IDを渡すも、システムのペナルティで「注目度ゼロ（消去）」となり処理落ちして消える。
* **カゲヤマ:** 独裁者の「声」を騙るプロパガンダ長官。城の疑似重力を維持するため、計算し尽くされた配信で大衆を操作する。
* **ゲン:** 空白の座標の境界に住む盲目のジャンク屋。アナログ機器を直し、世界が幻覚であることに気づきかけている老人。
