# 密室死のロジック（シンドウ事件）

### 事件概要
* **被害者:** シンドウ。元システムエンジニア。アテンション・エコノミーの設計思想に疑問を抱いていた異端者。
* **発見場所:** 中層「忘却の温室（日本区画）」内の電波暗室。完全密室。
* **遺体状態:** 極度の脱水によるミイラ化。外傷なし。死後推定2週間。

### 表向きの捜査（キジマが辿る道筋）
1. 密室に外部からの侵入痕跡なし。暗室の電磁遮蔽は完全。
2. 暗室内にImpの送受信記録なし。つまり「関心による攻撃」は不可能。
3. 毒物・外傷・窒息の痕跡なし。
4. 結論：原因不明の突然死。治安維持局は「電波暗室の危険性」を理由に事件を封印しようとする。

### 真相（段階的に明かされる）
1. シンドウは独自にARデバイスの存在に気づいた（ゲンとの接触が契機）。
2. 彼はデバイスの脱獄（ジェイルブレイク）コードを開発し、自らに適用した。
3. 安全装置（フェイルセーフ）を意図的に無効化し、電波暗室に入った。
4. 暗室進入の瞬間、デバイスからの全入力がゼロになった。
5. 30年以上にわたりデバイスに依存してきた神経系が、「全人類からの完全な断絶」と誤認。
6. 交感神経の暴走→心拍の異常亢進→心室細動→心不全。
7. 「アテンションの極限の離脱症状」による死。

### シンドウの遺したもの
* **手記:** 紙の手帳に記録された、デバイスに関する断片的な考察と脱獄コード。キジマが発見し、世界の真実を知る鍵となる。
* **脱獄コードの不完全さ:** シンドウのコードにはフェイルセーフの段階的解除機能がなかった。キジマが自らAR部品を物理的に破壊する方法を選ぶのは、この教訓から。
