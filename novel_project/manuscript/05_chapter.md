# 第五章　紙の手帳

　缶ビールの空き缶が六本になった。

　キジマは自室のテーブルでシンドウの手帳を読んでいた。事件から五日。日中は通常業務をこなし、夜になると手帳を開く。胃薬の消費が倍になった。缶ビールは晩酌の定量を超えている。

　手帳の中盤。シンドウの記述が、技術的なメモから個人的な独白に変わる場所。

「目を閉じて壁を触れ。お前の指が知っていることを、お前の目は知らない」

　キジマは手帳から目を上げ、自室の壁を見た。白い壁紙。均一な表面。目を閉じて手を伸ばし、壁に触れた。指先にざらつきがあった。壁紙のざらつきだと思った。思おうとした。目を開ける。白い壁紙。滑らかに見える。目を閉じる。ざらつく。

　手帳に戻った。

「全員が同じ嘘を見ている。嘘の中で生まれ、嘘の中で死ぬ。嘘を嘘と知る方法は一つ。切断」

「ゲンは最初から知っていた。あの人は見えないから。見えないことが、唯一の正気だった」

　ゲン。前にも出てきた名前。地図にない座標の人物。「見えない」とはどういう意味か。視覚障害か。しかし「唯一の正気」とは。

　ページをめくる。小さな図があった。人間の後頭部のスケッチ。シンドウは絵が下手だった。しかし位置関係は明確だった。首の付け根、右耳の後ろ、左耳の後ろ、後頭部の中央。四箇所に小さな円が描かれ、それぞれに番号が振られている。

　円から引き出された矢印に注釈。

「①右耳後部：聴覚補正モジュール。除去しても致死的ではない。音のバランスが崩れるだけ」
「②首の付け根：視覚サブプロセッサ。除去で視野の一部が剥離する。出血注意」
「③左耳後部：体性感覚フィルタ。触覚・痛覚・温度覚の調整。除去後、感覚が過敏になる」
「④後頭部中央：メインユニット。全機能の統合制御。これを抜けば完全停止」

　そして太字で。

「段階的に外せ。順番を間違えるな。④を先に抜けば心停止。①②③を除去し、最低六時間は部分停止に慣らしてから④。急ぐな。急げば死ぬ」

　キジマはページを凝視した。これは――何かの部品の除去手順。人間の頭部に埋め込まれた部品。聴覚補正、視覚サブプロセッサ、体性感覚フィルタ。

　手帳の前半の数式と回路図が、この部品に関するものだとすれば。シンドウは何かを「自分の体から外そうとした」。そして「一気にやった」ために死んだ。

　みぞおちの奥が冷たくなった。缶ビールの酔いが醒めていく。キジマは万年筆を取り、自分の手帳に書いた。

「シンドウは④を最初に抜いた？　安全装置を飛ばした？　なぜ？」

　答えは手帳にはなかった。シンドウの記述はこの図の後、急激に乱れていた。文字が大きくなり、行が斜めになり、判読不能な殴り書きが数ページ続く。最後の判読可能な一行。

「すまない。もう我慢できない。見たい。本当の」

　翌日。局。

　キジマはいつもどおりの時間に出勤し、いつもどおりの缶コーヒーを買い、いつもどおりのデスクに座った。灰色の均衡。しかし体の内側は均衡を失っている。手帳の内容が、内臓の裏側に貼りついて離れない。

　レイの視線を感じる回数が増えていた。

　レイは変わらず有能だった。別件の報告書を完璧に仕上げ、聞き取り調査のスケジュールを管理し、上層部との連絡を漏れなくこなしている。しかし、キジマが引き出しを開けるたびに、レイの視線が刺さった。引き出しに手帳はもう入れていない。自宅に移した。しかしレイはそれを知らない。

　昼前、レイがキジマのデスクに来た。

「シンドウ事件の報告書、まだ最終承認していないですよね」

「ああ。忘れてた」

「早めにお願いします。上が気にしてます」

　「上が気にしている」。レイの声は穏やかだったが、その一語の温度が低かった。上とは誰か。課長か。その上か。「気にしている」のは報告書の遅延か、それともキジマの動きか。

　キジマは引き出しから報告書を出し、「事故死」のスタンプを押した。レイの目の前で。インクが紙に染みるのを確認して、レイに渡した。

「これでいいだろう」

　レイは報告書を受け取った。頷いた。去った。

　レイが安心したのではないことを、キジマは知っていた。レイは「キジマが報告書にスタンプを押した」という事実を、上に報告するために去ったのだ。レイの足音が廊下を遠ざかっていく。規則正しい足音。制服に皺のない女。

　業務終了後、キジマは忘却の温室に向かった。

　二度目の訪問。封印処理は完了しており、暗室のドアには黄色いテープが貼ってある。しかしキジマの捜査官IDでアクセスコードを入力すれば、テープの下のロックは開く。

　今回は暗室には入らなかった。暗室の外壁を調べた。

　壁に手を触れた。

　視覚的には、壁は滑らかな白い樹脂パネルだった。継ぎ目は均一で、清潔な表面。施設の仕様書通りの内装材。

　しかしキジマの指先に伝わるのは、ざらついたコンクリートの感触だった。凹凸がある。冷たい。微かに湿気を含んでいる。目を閉じた。指先を滑らせた。骨材の粒が指紋に引っかかる。砂利とセメントの手触り。

　目を開けた。白い樹脂パネル。滑らかな表面。

　目を閉じた。コンクリート。

　シンドウの言葉が蘇った。「お前の指が知っていることを、お前の目は知らない」。

　キジマは壁から手を離し、手帳を出した。万年筆のキャップを外し、書いた。

「壁が嘘をついている。触覚と視覚が一致しない。どちらが本物か。指のほうだと思う」

　壁の角に、古い換気口があった。三十センチ四方の金属格子。視覚的には壁と同じ白い樹脂に見える。触ると、錆びた金属だった。鉄の格子。錆が指に付着する。赤茶色の粉。

　格子の隙間から風が来た。冷たい風。機械油の匂いを含んでいた。白い樹脂の壁から機械油の匂いはしない。この匂いの出所は、目に見えるものの中にはなかった。鼻だけが知っている。指だけが知っている。

　キジマは格子のスケッチを手帳に描いた。万年筆のインクが指に滲む。インクの感触。紙の繊維。鉄錆の粉。すべて指が知っていること。

　帰り道、キジマは自分が尾行されていることに気づいた。プロの尾行ではない。二十メートル後方に二人。歩調を合わせている。治安維持局の若手。たぶんレイの指示。

　キジマはいつもの道で帰った。いつもの店で缶ビールを買った。いつもの部屋に帰った。灰色の均衡。何も異常はない。そう見えるように、そう振る舞った。

　鍵をかけ、靴を脱ぎ、手帳を出した。最後のメモ。

「尾行あり。二名。レイか上か。手帳を移す。この部屋はもう安全ではない」

　キジマは手帳を閉じ、窓辺に立った。缶ビールを開ける。三丁目のビルはいつもどおり立っている。輪郭は跳ねていない。

　しかしもう、あのビルが本当にそこにあるのかどうか、キジマには確信が持てなかった。
