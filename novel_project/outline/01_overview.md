# 作品概要・全体プロット

### 1. プロジェクト定義
* **ジャンル:** ハードSF / ノワール・ミステリ / 群像劇
* **対象読者:** 複雑な社会構造の風刺、ハードボイルドな疲労感、哲学的な余韻を好む大人。
* **トーン＆マナー:** スティーヴン・キング的「泥臭い日常に侵食する狂気」と、フィリップ・K・ディック的「現実のゲシュタルト崩壊」。安易な能力バトルや感傷的なセリフ回しを完全排除した、静的でヒリヒリするリアリティ。

### 2. 根幹のテーマ
* **表層テーマ:** 「アテンション・エコノミー（注目経済）の物理化」。他者の関心（いいね、炎上、信仰）が質量や防壁といった物理法則を書き換えるディストピア。
* **深層テーマ:** 「虚構（加工された現実）からのログアウトと、等身大の現実（リアル）の受容」。
* **最大の隠し設定（※執筆AIへの絶対指示）:**
    * 「関心が物理法則を変える」というのは**完全な嘘**である。
    * 全人類の脳内には出生時からARデバイスが寄生しており、「関心に応じた視覚エフェクト、疑似重力、疑似痛覚」を脳に直接送り込み、**集団幻覚**を見せているだけ。
    * デバイスはインプレッションに応じて内分泌系（ドーパミンやアドレナリン）を直接コントロールし、全能感や質量を「錯覚」させている。
    * 執筆時はこの真実を隠し、序盤〜中盤は「本当に物理法則が歪んでいる」ように描写すること。ただし、人が消える際の「テクスチャのポリゴン化」や「足音の遅延」など、デジタルなバグ（処理落ち）としての伏線を徹底的に張ること。
