# 進行プロット（交錯する3つのルートと決着）

* **【第1幕〜第2幕】各々の必然の移動**
    * **ジンのルート（血塗られた上昇）:** シズカの死を越え、カゲヤマの予備IDで「重力の城」へ向かう。上昇の推進力を得るため、意図的にフェイクニュースで大衆を扇動し、自己嫌悪に苛まれながらエレベーターを昇る。
    * **恋人たちのルート（復讐の連鎖を断つ）:** シドウに追われ中層（デッド・トラフィック）へ落下。追跡でボロボロになったシドウを殺す好機を得るが、暴力のサイクルから降りるため「彼を殺さず」最下層へ向かい、ゲンと出会う。
    * **キジマのルート（アナログへの回帰）:** シンドウの手記から世界の真実を知り、体制側（レイ達）から追われる。追跡を逃れるため、自らの首の後ろを切り裂き、AR部品を物理的に一つずつ破壊（アンインストール）しながら、激痛と共に地下要塞へ降りていく。

* **【第3幕〜第4幕】大いなるログアウト（同時多発の決着）**
    * ジンが玉座に到達。そこには重圧に潰された哀れな老人が。ジンはカゲヤマの妨害を退け、老人の真の姿をアナログ回線で全世界へ無加工配信。「畏怖」が「失望」に変わり、世界規模のシステムエラー（フリーズ）が発生。
    * システムが停止し帝国軍のエイム補正や衝撃吸収がダウンした隙を突き、完全に「ただの中年男」に戻ったキジマが最深部へ。システムが張った「重さ1万トン」の最強のAR防壁（幻覚）を、彼だけが「ただの鉄のレバー」として引き下ろす。
    * 地球全土のARが沈黙。サイバーパンク都市の幻覚が消え、むき出しのコンクリートと真の重力が現れる。タリクとエララはノイズの消えた互いの「本当の顔」を見て、素手で抱きしめ合う。キジマは血まみれで、真実の夕焼けを見て微笑む。
